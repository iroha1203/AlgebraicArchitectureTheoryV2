# Output Reading Guide

Read current ArchSig output in this order:

1. `archsig-run-manifest.json`
2. `archsig-analysis-summary.json`
3. `archsig-insight-report.json`
4. `archsig-measurement-packet.json`
5. `archsig-atom-viewer-data.json`
6. Optional `archsig-comparison-report.json`
7. Optional `archsig-gate-report.json`

Keep all findings bounded by the selected inputs and source evidence.
