## 1. T1. 周期と Torelli: 本質の指紋

周期と Torelli の主題は、障害コホモロジーに刻まれた period / monodromy データを「本質の指紋」として読み直す試みである。古典 Torelli 定理が「周期写像は対象を復元する」と語ったように、ここでは選択された測定文脈のもとで、周期 pairing の値の束が refactor 同値で割った codebase essence を同定できるか、を問う。G4(周期 Stokes 会計)が順方向(障害から周期へ)の会計を担うのに対し、本テーマはその逆問題側——周期から本質へ——を扱い、輸送幾何(Wasserstein period・Kantorovich potential)へも橋を渡す。いずれの候補も context / detecting family / finite regime に厳密に相対化され、計測への適用は移植仮定付きである。決定するのは scheme 全体ではなく、選ばれた語彙内の障害構造あるいは essence stack に限定される。

### NT-01 Refactor 同変周期 Torelli 判定: 同変周期束が codebase essence を決定する 〔予想・予想(強)〕

- **主張**: 固定 (V, U, J, k) と finite measurement regime のもとで、codebase essence の quotient stack Ess_U(X) = [X^U / Ref_U](第VI部 定義14.1)を考える。U-detecting な部分族 Rep_det ⊂ Rep(AAT)(第VII部 定義15.3)を固定し、各 R ∈ Rep_det と各 1-cycle γ に対する strict obstruction period Per_{ω,γ}^{tr_U}(X)(第VII部 定義5.2)を、refactor groupoid Ref_U の作用で同変化した同変周期束 Per^{Ref}_U(X) として束ねる。予想は次を主張する。仮定として (a) Rep_det が selected Obs_U に U-conservative(定理15.4 の前提:U-adequate cover、witness/axis exactness、coefficient discipline)、(b) period pairing が Ref_U の作用と同変、(c) finite regime、を置く。このとき二つの architecture geometry X, Y について、同変周期束の同型 Per^{Ref}_U(X) ≅ Per^{Ref}_U(Y)(全 R ∈ Rep_det と全 cycle で値が一致し、かつ Ref_U-作用を保つ)が成り立つことと、Ess_U(X) ≅ Ess_U(Y)(refactor 同値として本質が一致)が成り立つこととは同値である。すべて固定された (Rep_det, U, J, k, cover) に相対化され、未選択 axis には決定主張をしない。

- **依拠**: 第VII部 定理6.1 Period Separation、定義5.2 Strict Period、定義15.3 U-Detecting Representation Family、定理15.4 Representation Conservativity、第VI部 定義14.1 Codebase Essence as Quotient Stack [X^U/Ref_U]、定理12.4 Operation-Invariant Galois Correspondence、第IV部 周期 pairing と Čech homology model。

- **非自明性**: 既存の Period Separation(定理6.1)は「異なる representation は分離しうる」という片方向の負の主張——同じ graph period でも semantic / effect period は異なりうる——にとどまり、第VII部の開放端は「period が geometry を決定するか=Torelli 型問題は未提起」と明記する。本候補はその converse を、単なる geometry ではなく refactor 同値で割った essence stack に対して、しかも U-detecting family の conservativity(定理15.4)を橋として立てる。Torelli 定理の AAT 類似であり、(i) 何を不変量とするか(同変周期束)、(ii) 何を決定するか(text identity でも graph iso でもない quotient stack essence)、(iii) 決定が成り立つための detecting / conservative 条件、の三点を新たに結ぶ。G1–G8(計測の忠実性・容量・分解)とも既存の holonomy / monodromy 予想(経路依存 debt)とも異なる、同一性判定型の命題である。

- **CS/SWE帰結**: 「二つのコードベース(あるいはリファクタ前後)は本質的に同じか」を、テキスト一致でもグラフ同型でもなく、選んだ検出表現族の周期(障害 cohomology class と cycle の pairing 値の束)が refactor 同変に一致するか、という計算可能なチェックに還元する。具体的には、(1) リファクタ正当性の検証——リファクタが essence を保ったか否かを全 detecting representation の周期保存で判定し(第VIII部 定理7.3 Refactor Invariance の周期版拡張)、(2) コードベース系統(fork / clone)の同一性判定——graph iso では誤検出する「見た目は同じだが意味 / 効果が違う」ケースを semantic / effect period で分離しつつ本質同型を周期束同型で確定し、(3) detecting family の最小性——essence を決定するのに必要十分な representation の組(どの解析窓を揃えれば本質が確定するか)を Torelli 性から逆算する。これは「リファクタ後も本質が同じことの証明書」という肯定的同一性 claim を、移植仮定付きで初めて可能にする。

- **証明/根拠の見込み**: 逆向き(Ess ≅ ⇒ 周期一致)は易しい。refactor 同値が selected invariant と obstruction sheaf を保つ(定義14.1 の groupoid data が law / obstruction / signature / structure sheaf と整合する)ため、同変周期は essence の不変量として well-defined になり一致する。順向き(周期一致 ⇒ Ess ≅)が予想の核である。U-conservative family では R(α) = 0 が α = 0 を導く(定理15.4)ので、周期束が分離する class の情報から obstruction sheaf を係数まで復元でき、復元された sheaf 上の Ref_U-作用の同変構造が essence stack の atlas / cocycle data を決める——ここが Torelli の「周期 → 対象」復元に相当し、最も非自明な部分である。証拠として、Period Separation(定理6.1)が示すのは「単一表現では不十分」であり、これは detecting family を太らせる必要性という本候補の動機を対偶的に支持する。最小例として、付録 B の擬円周モデルでは graph period は一致するが strict period(B.6 の ⟨ω,γ⟩ = 1)が分離する二つの geometry が、semantic + effect period まで含めると essence で区別 / 同定される。未証明部分は、周期束からの sheaf 復元の関手性、無限族ではなく有限 detecting family での決定性、quotient stack の 2-同型レベルでの再構成である。

- **関連**: G4、G3、第VI部 Ess_U、第VII部 U-detecting family。

### NT-02 輸送 Torelli: 負債輸送スペクトラムによる architecture geometry の分離 〔予想・予想(探索的)〕

- **主張**: 固定 measurement profile クラス M と representation family Rep に対し、architecture geometry X の輸送周期スペクトラムを TransSpec_M(X) := { W_1(Ω_U(X), Ω_U(op(X))) : op ∈ 選択された operation generator 族 } と、各 op に付随する Kantorovich potential φ*_op(候補2)の組として定義する。予想は次を主張する。Rep が selected obstruction family に U-detecting(定義15.3)であり、M が finite measurement regime であり、operation generator 族が transport-faithful(各 op の輸送 potential が support を分離する)であるとき、TransSpec_M(X) = TransSpec_M(Y)(輸送距離の多重集合と potential の緊張辺パターンが一致)ならば、X と Y は selected geometry 内で refactor-equivalent(第VI部 Ref_U(X) の射で結ばれる)である。逆に refactor-equivalent な X, Y は TransSpec_M が一致する(定理7.3 から従う易方向)。すなわち、輸送周期スペクトラムは selected geometry を決定する完全座標系である。すべて M / Rep / operation 族に相対化され、未選択 axis・未計測 op には分離主張をしない。

- **依拠**: 第VII部 定理6.1 Period Separation、定義5.2 Strict obstruction period、定義15.3 U-Detecting Representation Family、定理15.4 Representation Conservativity、第VI部 定義12.1 Refactor groupoid Ref_U(X)、定理12.4 Operation-Invariant Galois Correspondence、候補2(Kantorovich 双対 potential)、候補1(輸送計量)、第VIII部 定理7.3 Refactor Invariance under Equivalence(易方向の母体)、および第VII部 開放端「period が geometry を決定するか=Torelli 型問題が未提起」。

- **非自明性**: 第VII部の開放端は「どの representation family が geometry を分離する完全座標系になるか(Torelli 型問題)が未提起」と明示しており、本予想はそこに直接住む。Period Separation(定理6.1)は「同じ broad period でも別 representation で異なりうる」という分離の存在を述べるだけで、逆向き(period が一致すれば geometry が一致する)という Torelli の核心は扱っていない。輸送スペクトラム(距離 + potential)を座標系に用いる点が新しく、古典 Torelli(周期写像の単射性)の AAT 類似を Wasserstein period で実現しようとする。U-detecting かつ transport-faithful という相対化された十分条件下での単射性を狙う野心的予想である。

- **CS/SWE帰結**: 「二つのコードベース(あるいは同一コードの二版)が、選ばれた幾何の意味で本質的に同じ設計か」を、輸送周期スペクトラムの一致で判定する道を開く。これは graph isomorphism(粗すぎる)でも text identity(細すぎる)でもない、refactor 不変な architecture essence(第VI部 quotient stack)の計算可能な代理座標を与える(逆向き = 易方向は定理7.3 で保証済み、難方向 = 分離が予想)。実務的には「この大規模リファクタは本当に等価変形だったか、それとも幾何を変えたか」を輸送スペクトラムの差として検出し、回帰の有無を essence レベルで監査する(移植仮定:U-detecting representation・transport-faithful operation 族・finite regime を固定した profile 内)。

- **証明/根拠の見込み**: 支持証拠は二段にある。(1) 易方向(refactor-equiv ⇒ 同スペクトラム)は第VIII部 定理7.3(class 零性保存)と候補1(計量が pushforward で保存される)からほぼ証明可能である。(2) 難方向の特別な場合として、monomial / square-free regime では Ω_U が Stanley–Reisner 複体(定理5.6C)の faces 質量で決まり、輸送 potential の緊張辺が forbidden support パターンを復元する。Alexander dual(定理5.2)と組めば minimal forbidden support 族が輸送スペクトラムから読めるので、SR 複体が一致 ⇒ I_Ob が一致 ⇒ geometry 一致、という鎖で特別な場合に Torelli が成立する見込みである。一般の難方向は U-detecting 仮定(候補2 の双対 no-op 検出を全 generator op に適用すると Ω_U(X) が op 作用ごと復元される)が鍵となる。難所は、operation 族が geometry の全自由度を揺さぶる完全性(transport-faithful)の特徴づけであり、これが予想に留まる本質的理由である。古典 Torelli が曲線・アーベル多様体でのみ成立する事情と平行する。

- **関連**: G4、NT-01、NT-43。

### NT-03 period family の忠実性と分離下界: 周期は障害コホモロジーを決めるが scheme は決めない 〔予想・予想(強)〕

- **主張**: 固定 analytic reading context(AtomVocabulary V、LawUniverse U、CoverageTopology J、係数、representation family Rep、period target Λ)を相対化基底とする。architecture scheme X, Y が同じ context を共有し、両者が finite measurement regime にあるとする。strict obstruction period の全族 Per(X) := { Per_{ω,γ}^{tr_U}(X) | ω ∈ H^n(X, Ob_U), γ ∈ H_n(X, ℤ), n ≥ 0 } を period family と呼ぶ。(Torelli 忠実性予想)Rep が U-detecting(定義15.3)かつ cover が U-adequate かつ axis / witness exactness が成立する regime において、period family の同型 Per(X) ≅ Per(Y)(全 obstruction class と全 cycle の pairing 値が period-preserving 写像で一致)が成り立つならば、X と Y は obstruction-cohomology レベルで同型である:すべての n で H^n(X, Ob_U) ≅ H^n(Y, Ob_U) が period pairing と compatible に成立する。(分離下界・部分定理)逆に、ある strict period が異なれば(∃ ω, γ で Per_{ω,γ}(X) ≠ Per_{ω,γ}(Y))、X と Y は obstruction-cohomology 同型ではない。結論は context / detecting family / exactness に相対化され、geometry 全体(scheme structure)の同型までは主張しない——period は cohomology を決めるが scheme を決めるとは限らない、という claim boundary を保つ。

- **依拠**: 第VII部 定義5.2 Strict obstruction period Per_{ω,γ}^{tr_U}、定義15.3 U-Detecting Representation Family、定理15.4 Representation Conservativity under Adequacy、定理6.1 Period Separation、および第VII部 開放端「period が geometry を決定するか(Torelli 型問題)が未提起」。

- **非自明性**: 第VII部の開放端は「Period Separation の量的精密化が無い。どの representation family が geometry を分離するための完全な座標系になるか(period が geometry を決定するか=Torelli 型問題)が未提起」と明示している。定理6.1(Period Separation)は「異なる reading で period が異なりうる」という分離可能性のみを述べ、その逆——period が一致すれば同型——という忠実性(Torelli の核心)は誰も述べていない。本候補は標準代数幾何の Torelli 定理(period が対象を決定する)の AAT 類似で、定理15.4(U-detecting family の conservativity)を period-pairing レベルへ持ち上げる。U-detecting + exactness を仮定すると period family が H^n(X, Ob_U) の完全な不変量になるという主張は深く、既存の boundary holonomy 等の予想や G1–G8 とは全く重ならない。

- **CS/SWE帰結**: 「二つのコードベース(または同一コードベースの二バージョン)が architecture 障害として本質的に同じか」を period family の比較だけで判定できる、新しい同値判定を与える。これまで architecture の同値は「text identity は厳しすぎ、graph isomorphism は粗すぎ」(原則14.2)で適切な不変量が無かったが、period family は (i) U-detecting family 下で obstruction cohomology を完全に決める忠実な不変量であり、(ii) 一つでも period が違えば非同型という分離下界を持つ。実務では「リファクタ前後で period family が保たれたか」で「障害構造を保つ安全なリファクタか」を判定でき(第VIII部 定理7.3 Refactor Invariance の period 版)、「どの period が変わったか」で変化した障害軸を局在化できる。予想段階のため計測適用は移植仮定付きである(detecting family の存在・構成が前提)。

- **証明/根拠の見込み**: 支持証拠は三つある。(1) 第VII部 定理15.4 は既に「U-detecting + U-adequate + exactness なら ∀R ∈ Rep, R(α) = 0 ⇒ α = 0」(conservativity)を与えており、period pairing が R(α)(cohomology class の reading)の一形態であることから、conservativity を pairing values の族へ拡張すれば忠実性が出るのが自然な経路である。(2) 有限 regime では H^n(X, Ob_U) は有限次元 k-ベクトル空間であり、period pairing H^n × H_n → Λ は cohomology と homology の双対 pairing(普遍係数定理)であって、非退化なら period values が cohomology class を座標づける(双対基底の存在)。detecting family がこの非退化性を供給するという仮定である。(3) 例5.2B(擬円周 strict period = r_sync − r_async)では period 値 1 が H^1 ≅ ℤ の generator を一意に決める具体例があり、最小ケースで Torelli が成立している。逆向き(分離下界)は対偶により定理6.1 の精密化として直ちに従う。一般の scheme structure までの Torelli が成り立たない理由——period は cohomology を決めるが decoration / scheme atlas は決めない——を claim boundary として明示し、過大主張を避ける。予想として残す主因は、detecting family の存在 / 構成定理(第VII部 開放端の universal / terminal family)が未確立である点にある。

- **関連**: G4、NT-01、第VIII部 定理7.3。

## 2. T2. 特異点・半安定還元・標準形

T2 のテーマは、God object を「直しにくい一点」ではなく特異 stratum として代数的に位置づけ、その解消・収束・残債を双有理幾何の語彙(blow-up 列、半安定還元、discrepancy 会計、log-canonical 閾値、接錐、Milnor 数)で構造化する試みである。リアーキテクチャの問いは「直せるか」だけでなく「有限手数で解消できるか」「どこに不可避の負債が局在して残るか」「その複雑性に整数下界はあるか」へと精密化される。以下の候補群は、特異点を refinement で半安定化して monodromy を冪単へ正規化する(NT-04, NT-05, NT-10)、normal cone 沿いの blow-up 列で smooth 終端へ解消し残差地図を描く(NT-06)、接錐・Milnor 数・discrepancy/lct という整数・有理数不変量で複雑性と予算を会計する(NT-07, NT-08, NT-09)の三系統に対応する。いずれも固定パラメータ p・選ばれた regime・移植仮定に相対化された命題であり、その範囲内でのみ語る。

### NT-04 障害の半安定還元と God object 標準形: refinement による monodromy 冪単化と monodromy-weight 対応 〔予想・予想(探索的)〕

- **主張**: 固定パラメータ p のもとで、X が U-singular stratum S(第VI部 定義5.1: ある first-order lawful deformation η で ob_U(η)≠0)を持ち、operation loop γ に沿う Gauss-Manin monodromy ρ_U^{GM}(γ) ∈ Aut(H^n(S, Ob_U))(第VI部 定義10.2)が一般には非冪単(固有値が 1 でない、あるいは非自明な Jordan 構造をもつ)とする。予想として、適切な admissible cover refinement(第II部 J_U の base-change で得られる finite witness-closure cover の細分)π: X' → X が存在し、(i) 半安定還元: S の引き戻し S' 上で monodromy ρ_U^{GM}(γ) が冪単((ρ − id)^N = 0 となる N が存在)になる、(ii) God object 標準形: refinement 後の S' が「複数 lawful loci の正規交叉(normal-crossing)的交わり」として表示でき、その交わりの組合せ構造(第V部 LawConflict_i の minimal forbidden support の simplicial 構造)が monodromy の冪単部分 N = ρ − id を完全に決定する(monodromy-weight 対応)、(iii) refinement に相対化した monodromy debt(拡張ノートの μ_x / AMI)が冪単部分の不変量(N の rank・nilpotency index)として読める、が成り立つ。すべて選択された p / operation family / coefficient / refinement に相対化する。
- **依拠**: 第VI部 定義5.1 U-Singular Stratum と定理6.1 Architecture Singularity Criterion、定義10.2 Finite Gauss-Manin system / Gauss-Manin monodromy、定理11.1 Monodromy Debt、第V部 定理5.6C Stanley-Reisner Obstruction と定義5.4 Monomial Law Conflict、第II部 定義7.1 AAT Grothendieck Topology(refinement / base-change)、拡張ノートの計測モノドロミー μ_x / AMI。
- **非自明性**: AAT には singularity(第VI部)、monodromy(定理11.1)、LawConflict の simplicial 構造(第V部)が個別に存在するが、それらを「特異点を refinement で半安定化し monodromy を冪単へ正規化する」一本の定理で結ぶものは無い。本予想は半安定還元定理(Mumford ほか)と monodromy-weight 予想(Deligne)の AAT 類似として、(1) 非冪単 monodromy が refinement で冪単化できるという正規化、(2) 冪単部分が LawConflict の正規交叉組合せ構造で決まるという対応、の二段を主張する。これは God object(複数 law loci の非横断交叉、第VI部 定義7.1)を blow-up / refinement で正規交叉形へ分解する標準形を与え、第VI部の「operation homotopy が選択依存(W2)」「non-abelian H^2 の精密化が空白」という開放端へ直接切り込む。既存対象の上に純粋に構成される点で自明な系ではない。
- **CS/SWE帰結**: (a) 高次障害が集中した特異 stratum(直しにくい God object)を、どう refinement(= モジュール再境界化)すれば monodromy debt が制御可能な冪単=段階的に解消可能な形になるか、というリアーキテクチャの標準形を与える。冪単 monodromy は「有限回の局所修正の合成で恒等へ戻せる」ことを意味するため、半安定還元は「この God object はこの細分のもとで原理的に段階解消可能」という肯定的 certificate になる。(b) monodromy-weight 対応により、history-dependent debt(同じ endpoint でも経路で残る負債、定理11.1)の冪単部分を LawConflict の組合せ(どの forbidden support が交わるか)から予測でき、どの操作順序が path-dependent 残留を生むかを交叉組合せとして会計できる。(c) 予想段階かつ refinement 相対であり、refinement の存在自体が claim boundary をなす(全 God object が半安定化するとは限らない)。移植仮定: Gauss-Manin の存在(第VI部 10.1 は自動でない)と operation groupoid の固定。
- **証明/根拠の見込み**: (1) square-free monomial regime(第V部 定義5.4)では LawConflict が Stanley-Reisner 複体の組合せで完全記述され(定理5.6C)、この regime の monodromy は forbidden support の lcm-lattice 上の置換作用として有限位数または冪単に分解する(monomial ideal の Tor は多重次数付きで Jordan 構造が読める)ので、(ii) の monodromy-weight 対応が monomial 特別ケースで成立する見込みがある。(2) 半安定還元の古典的核(有限拡大 / blow-up で monodromy を冪単化)は finite poset site の有限被覆細分(第II部 finite witness-closure cover)へ組合せ的に翻訳でき、有限性から N が冪零(有限次元上の準冪単作用素の冪単部分)になることが期待される。(3) 定理11.1 が既に「non-trivial monodromy = hidden debt」を確立しており、本予想はその debt を冪単正規形へ分解する次の階層に当たる。反証されうる点: refinement で冪単化できない本質的に wild な monodromy(非可換 H^2 / gerbe 層、第VI部 定義16.1)が存在する場合、予想は「半安定化可能な stratum クラス」へ制限される。
- **関連**: 第VI部 monodromy、NT-25、第V部 LawConflict。

### NT-05 migration の半安定還元: temporal 障害の冪単化と breaking change の正規化 〔予想・予想(探索的)〕

### NT-06 アーキテクチャ半安定修復解消: 特異 stratum の normal cone 沿い有限 blow-up と残差地図 〔予想・予想(強)〕

- **主張**: 固定 p、effective deformation-obstruction theory regime、well-founded repair comparison profile を仮定する。U-singular stratum S(ある first-order lawful 変形 η で ob_U(η) ≠ 0 ∈ H^1(T_{S/U}))に対し、geometric repair(architecture morphism)の有限列 r_N ∘ … ∘ r_1: X_N → … → X_0 = X であって、(1) 各 r_k が normal cone 方向(C_{S/Flat_U} 内の I_U 消去方向)に沿う blow-up 型 repair、(2) comparison profile の下で各ステップが strict decrease、(3) 終端 X_N が U-smooth(全 deformation test で ob_U = 0)、(4) 例外 locus(repair が触れた support)が transferred obstruction をもつ normal-crossing 型 boundary に制御される、を満たすものが存在すると予想する。さらに半安定形として、終端で残る derived conflict LawConflict_i は normal-crossing 例外 locus に局在し、それ以上の repair では消えない不変核(scale-stable debt 的残差)をなす。すべて選ばれた profile / regime に相対化する。

- **非自明性**: 第VI部は singularity(repair 困難点)を定義し criterion を与えるが、「特異点を有限手数で解消して smooth model に到達できるか」という解消定理は完全に欠落している(開放端に「normal cone を計算する具体手段や deformation to normal cone の定理が無い」と明記)。第V部 定理13.3 / 13.4 は repair の停止性と健全性を与えるが、終端が U-smooth(特異性解消)に到達することは保証せず、終端の lawful 性は外部委譲されている。本候補は両者を橋渡しし、「termination + soundness ⇒ smooth 終端 + 制御された例外 locus」へ昇格する。特異点解消(Hironaka)と半安定還元の AAT 版であり、normal cone 方向の blow-up という幾何的 repair の意味づけ、例外 locus の normal-crossing 制御、残差の不変核という三層が非自明である。単なる「直せば良くなる」ではなく「どこに不可避の負債が局在して残るか」を解消の幾何で特定する。

## 3. T3. モノドロミー・Lefschetz・力学系

## 4. T4. 組合せ可換代数: 障害イデアルの深層

AAT 障害イデアルは、これまで「禁止パターンが何個あるか」という生成元の数(violation count)までしか測ってこなかった。だが Stanley-Reisner 環としての obstruction ideal は、minimal free resolution・局所コホモロジー・Castelnuovo-Mumford regularity・Tor 長といった、サイズに還元できない深い構造量を抱えている。本テーマは Hochster 公式・Reisner 判定・Eagon-Reiner / Terai 双対・Auslander-Buchsbaum-Serre 正則性規準・代数的離散 Morse 理論を AAT 障害イデアルへ移植し、冗長度・修復計画の並列不能性・lawful locus の代数的良性・破壊的変更の最小手数を、選ばれた reading と square-free witness regime に相対化した数学命題として定式化する。いずれも有限 regime では Macaulay2 で Betti 数・depth・regularity まで実際に検算でき、計測接続は移植仮定付きで tooling 側へ渡す。共通の規律は、選ばれた語彙の内側でだけ語り、regime 外の witness/axis には何も主張しないことである。

### NT-17 Hochster-Betti 冗長制約定理: minimal free resolution が制約の冗長度を局在化する 〔定理候補・証明可能見込み〕

- **主張**: 固定 reading $p=(V,U,J,k)$ の square-free witness regime(第III部 定義5.6B)を仮定し、witness 変数集合 $E_W$、forbidden support 族 $\mathrm{Forb}_U(W)$、SR-ideal $I_{\mathrm{Ob}}^U(W)=I_{\Delta_U}(W)\subset R_W=k[E_W]$ をとる(定理5.6C)。$R_W/I_{\mathrm{Ob}}^U(W)$ の minimal graded free resolution の $i$ 番目の multigraded Betti 数を $\beta_{i,S}$($S\subseteq E_W$ は multidegree)とおくとき、次が成り立つ。
  - (A) [Hochster 公式] $\beta_{i,S}\bigl(R_W/I_{\mathrm{Ob}}^U(W)\bigr)=\dim_k \tilde H_{|S|-i-1}(\Delta_U|_S;\,k)$。ここで $\Delta_U|_S$ は witness 集合 $S$ 上の induced 部分複体(= $S$ に制限した admissible witness support の族)、$\tilde H$ は reduced simplicial homology。
  - (B) [冗長度=射影次元] $\mathrm{pdim}_{R_W}\bigl(R_W/I_{\mathrm{Ob}}^U(W)\bigr)=\max\{\,i\mid \exists S,\ \beta_{i,S}\neq 0\,\}=:\mathrm{RedDepth}_U(W)$ と定義すると、これは「forbidden pattern を切り出す制約族が、一次の独立 witness を超えて何段の syzygy(冗長関係の関係)を持つか」を局在化する。Hilbert syzygy 定理により $\mathrm{RedDepth}_U(W)\le |E_W|$。
  - (C) [Auslander-Buchsbaum 会計] $\mathrm{RedDepth}_U(W)=|E_W|-\mathrm{depth}_{R_W}\bigl(R_W/I_{\mathrm{Ob}}^U(W)\bigr)$。
  - 相対化: 結論はすべて選ばれた $V/U/J/k$ と square-free witness regime に相対化され、regime 外の witness/axis には $\beta$ も depth も主張しない。

- **非自明性**: G5(Hilbert 級数 Euler 恒等式)は二つの law universe の Tor 質量の交代和を読むが、単一 obstruction ideal の minimal free resolution の各 Betti 数を induced 部分複体の reduced homology へ完全に同定すること(Hochster 公式)は AAT 本文・拡張ノート・既存予想のどこにも無い。系5.6D は minimal generators(= irreducible witness)までしか読まず、higher syzygy(generator 間の冗長関係、さらにその関係)を一度も定式化していない。射影次元・depth・Auslander-Buchsbaum 会計は obstruction ideal sheaf に対して未着手であり、定義の言い換えではなく minimal resolution の存在と一意性(可換代数の非自明定理)に依拠する。
- **CS/SWE帰結**: 従来の lint/policy は「禁止パターンが何個あるか(generator 数=violation count)」までしか測れなかった。本定理は「制約族がどれだけ冗長か」を $\beta_{i,S}$ の higher $i$ 成分として初めて数値化する。$\beta_1$(generator)=禁止パターン、$\beta_2$=禁止パターン同士の重複(同じ witness を共有する forbidden support の syzygy)、$\beta_i$=$i$ 段重なった冗長制約、と読める。$\mathrm{RedDepth}_U(W)$ は「統合レビューで independent に潰せず、連鎖して効いてくる制約の段数」=どこまで遡って修正の波及を追うべきかの上界を与える。Hochster 公式により各冗長度は特定 witness 集合 $S$ 上の induced 部分複体のホールに局在するので、レビュー対象を「どの witness クラスタの何段目の冗長か」まで絞れる。depth が低い(= RedDepth が高い)ほど制約族が深く絡み合っており、局所修正が困難であることを Auslander-Buchsbaum 会計が定量保証する。

### NT-18 AAT Hochster 公式: 局所コホモロジーの多重次数分解と link の組合せ位相 〔定理候補・証明可能見込み〕

- **主張**: square-free witness regime で obstruction ideal $I_{\mathrm{Ob}}^U=I_{\Delta_U}\subset R_W=k[E_W]$ を固定し、law algebra structure sheaf の chart 環として商環 $S_W:=R_W/I_{\Delta_U}$(= lawful locus $\mathrm{Flat}_U(W)$ の座標環)をとる。$S_W$ の多重次数付き局所コホモロジー $H^i_{\mathfrak m}(S_W)$ の Hilbert 級数は、$\Delta_U$ の各 face $\sigma$(= 同時に許容される witness support、系5.6D の admissible witness support)の link $\mathrm{link}_{\Delta_U}(\sigma)$ に対する reduced simplicial 障害コホモロジー $\dim_k \tilde H^{i-|\sigma|-1}(\mathrm{link}_{\Delta_U}(\sigma);k)$ の和として閉形式で表される(Hochster の局所コホモロジー公式の AAT 版):
  $$\mathrm{Hilb}\bigl(H^i_{\mathfrak m}(S_W);t\bigr)=\sum_{\sigma\in\Delta_U}\dim_k \tilde H^{\,i-|\sigma|-1}\bigl(\mathrm{link}_{\Delta_U}(\sigma);k\bigr)\cdot\prod_{e\in\sigma}\frac{t_e^{-1}}{1-t_e^{-1}}.$$
  相対化: 選ばれた witness regime / 定数係数 $k$ / chart $W$ に相対化される。link は系5.6D の context refinement / boundary-local reading に対応する。
- **依拠**: 第III部 定理5.6C(Stanley-Reisner Obstruction Theorem)、系5.6D(Obstruction Invariants: faces = admissible support, links = context refinement)、定義7.1(Lawful Locus $\mathrm{Flat}_U(X)=V(I_{\mathrm{Ob}}^U)$)、第IV部 意味4.2(Cohomological Degrees)、拡張ノート B1(Hochster 公式、スポークとして名指しのみ)。
- **非自明性**: 拡張ノートの橋マップは Hochster 公式を B1 スポークとして名指しするだけで、AAT 定理化されていない。本候補は「lawful locus 座標環の局所コホモロジー」という代数的に深い不変量を、$\Delta_U$ の face ごとの link の reduced 障害コホモロジーへ完全分解する。これは G5(Hilbert 級数干渉 = Tor 質量の交代和)とは別物で、G5 が二法則の Tor 交代和を勘定するのに対し、本候補は単一 law universe の lawful locus の Cohen-Macaulay 性・depth・regularity を link の組合せ位相で読む。link = context refinement という系5.6D の reading に厳密な代数的意味(局所コホモロジーへの寄与)を与える点が新しく、第III部と第IV部を Hochster 公式で接続する橋になる。
- **CS/SWE帰結**: lawful locus 座標環の depth(= Cohen-Macaulay までの距離)を、各 admissible witness support $\sigma$ の link の reduced 位相だけから判定できる新しい道が開く。CS 読み: $\mathrm{link}_{\Delta_U}(\sigma)$ が非自明な $\tilde H$ を持つ最小次数が、support $\sigma$ を固定した(= その設計要素群を保持したまま)context refinement で初めて現れる「隠れた統合障害の最小次元」を与える。これは「どの設計要素の組を固定すると、どの次元の障害が局所的に立ち上がるか」という、従来 measure できなかった boundary-local な障害局在を組合せ計算へ落とす。移植仮定付き計測接続として、Macaulay2 の局所コホモロジー / link 計算でツール出力可能(B1 で Macaulay2 予告済み)。verdict 規律: link が acyclic な次数では unmeasured ではなく measured_zero の structural reading とする。
- **証明/根拠の見込み**: AAT 側の仮定($I_{\mathrm{Ob}}^U=I_{\Delta_U}$、$S_W=R/I_{\Delta_U}$、定数係数、square-free)を満たせば $S_W$ はちょうど Stanley-Reisner 環であり、古典的 Hochster の局所コホモロジー公式(Stanley『Combinatorics and Commutative Algebra』定理 II.4.1、Bruns-Herzog 5.3.8)がそのまま適用できる。AAT の仕事は三点: (i) 第III部 定理5.6C により obstruction ideal が SR ideal であることの確認、(ii) face = admissible support / link = context refinement の AAT reading(系5.6D)を Hochster 公式の組合せ項へ対応づけること、(iii) reduced simplicial cohomology を第IV部の定数係数障害コホモロジーと同一視する regime 固定。証拠: 最小例 $\Delta_U$ に頂点 $\{p,q,r\}$、$\mathrm{MinForb}=\{\{p,q\},\{q,r\}\}$(付録B の $I_{\mathrm{Ob}}=\langle pq,qr\rangle$)では $\mathrm{link}_{\Delta_U}(q)$ 近傍が非連結となり $\tilde H^0\neq 0$ が $q$ に局在する。これは付録B で $q$ が共有 witness factor として conflict を担う事実(B.5/B.8.4)と一致する。

### NT-20 AAT Reisner 判定: lawful locus の Cohen-Macaulay 性と低次障害消滅の同値 〔予想・予想(強)〕

- **主張**: square-free witness regime で lawful locus 座標環 $S_W=R_W/I_{\Delta_U}$ を固定する。次の三条件が同値であると予想する(Reisner 判定の AAT 版)。
  - (A) $S_W$ が Cohen-Macaulay(代数的: $\mathrm{depth}=\dim$)。
  - (B) $\Delta_U$ の各 face $\sigma$(空集合を含む)に対し、$\mathrm{link}_{\Delta_U}(\sigma)$ の reduced 定数係数障害コホモロジーが top 次数未満で消滅する: $\tilde H^i(\mathrm{link}_{\Delta_U}(\sigma);k)=0$ for all $i<\dim\mathrm{link}_{\Delta_U}(\sigma)$。
  - (C) すべての admissible witness support $\sigma$ について、$\sigma$ を固定した context refinement では top 次数を除いて局所-大域貼り合わせ障害が立たない(第IV部の link-relative $H^i$ が低次で消える)。
  - 相対化: 選んだ witness regime / 係数体 $k$ に相対化。古典 Reisner と同様、$k$ への標数依存があり得ることを明示する。
- **依拠**: 第III部 定理5.6C(Stanley-Reisner Obstruction Theorem)、系5.6D(links = context refinement)、第IV部 意味4.2(Cohomological Degrees)、AAT Hochster 公式(本提案 NT-18)、定義7.1(Lawful Locus)。
- **非自明性**: AAT には「lawful locus が代数的に良い(Cohen-Macaulay / depth が高い)とはどういう構造か」を判定する定理が一切ない。本予想は Reisner の古典判定を AAT へ移し、lawful locus の代数的良性を「すべての context refinement で低次障害が消える」という第IV部的・組合せ位相的条件に翻訳する。これは消滅定理(Kodaira / Grothendieck 型)の組合せ版に相当し、要求にある「標準 AG の強定理の AAT 類似」に直接応える。G1〜G8 や既存予想はいずれも障害の存在・分解・会計を扱うが、障害が構造的に低次で消える regime の特徴づけは未着手。前候補 Hochster 公式を前提に depth を link 位相で読み切る点で深い。
- **CS/SWE帰結**: 「この lawful locus(= この設計の許容構成空間)は、どの設計要素部分集合を固定しても、最上次数を除いて統合障害が立たない」という強い肯定的構造保証(Cohen-Macaulay = 障害が一様に最上次に集中)を組合せ判定できる新しい道が開く。CS 読み: Cohen-Macaulay な architecture は「局所レビューの十分性が次数横断的に成り立つ良い分割」であり、G2(c) の森条件($b_1=0$ の $H^1$ 消滅)を全次数へ一般化した「良い分割の代数的証明書」を与える。逆に非 CM なら、どの context refinement で低次障害が現れるか(= どこに本質的に厄介な貼り合わせが潜むか)を link の reduced 位相で局在化できる。移植仮定付き計測接続として、Macaulay2 の depth / Reisner 判定で verdict 化し、標数依存の警告を packet に明示する。
- **証明/根拠の見込み**: AAT 仮定($I_{\mathrm{Ob}}^U=I_{\Delta_U}$、定数係数、square-free)の下では $S_W$ はちょうど Stanley-Reisner 環で、古典 Reisner 判定(Reisner 1976; Stanley II.4, Bruns-Herzog 5.3.5: SR 環の CM 性 $\Leftrightarrow$ 全 link の reduced cohomology が top 未満で消滅)が文字どおり適用できるはずである。残るのは (i) 条件 (B) の link reduced cohomology を第IV部の link-relative 障害コホモロジー(条件 C)と同一視する regime の整合(前候補 Hochster 公式が分解項として同じ link cohomology を出すことが整合の証拠)、(ii) 標数依存の扱い。予想とした理由は、AAT の第IV部 $H^i$ を simplicial link cohomology へ忠実に落とす cover / 係数の固定が複数あり、どの固定でも同値が保たれるかの確認が必要なためである。特別な場合の証拠: $\Delta_U$ が shellable(例えば forest 状 nerve に対応する場合)なら CM が従い、G2(c) の局所レビュー十分性と整合する。
- **関連**: G2、NT-18、第IV部 link-relative $H^i$。

### NT-21 障害コホモロジーの組合せ Alexander 双対: 修復計画の regularity 下界 〔定理候補・要設計〕

- **主張**: square-free witness regime で obstruction ideal $I_{\mathrm{Ob}}^U=I_{\Delta_U}$ と、その Alexander dual ideal $I_{\Delta_U}^\vee$(定理5.2 の minimal repair hitting set を生成元とする ideal)を固定する。次が成り立つと予想する(Eagon-Reiner / Terai の AAT 版)。
  - (1) lawful locus 座標環 $R/I_{\Delta_U}$ の projective dimension は Alexander dual ideal の Castelnuovo-Mumford regularity に等しい: $\mathrm{pd}(R/I_{\Delta_U})=\mathrm{reg}(I_{\Delta_U}^\vee)$。
  - (2) repair-hitting-set ideal $I_{\Delta_U}^\vee$ が線形分解を持つ $\Leftrightarrow$ $R/I_{\Delta_U}$ が Cohen-Macaulay。
  - (3) 帰結として、minimal repair hitting sets(= $I_{\Delta_U}^\vee$ の極小生成元)の最大 support サイズと、それらの線形 syzygy 構造が、障害コホモロジーが非零となる最高次数(= 第IV部の最高 coherence 次数)の下界を与える。さらに $I_{\Delta_U}^\vee$ の極小自由分解は代数的 Morse 縮約(NT-24)で計算する。
  - 相対化: square-free regime / 係数 $k$ に相対化($\mathrm{reg}$ は標数依存し得る)。
- **依拠**: 第VIII部 定理5.2(Stanley-Reisner / Alexander Dual Repair Theorem)、原則5.3(Repair Hitting Set Is Not Repair Semantics)、第IV部 意味4.2(Cohomological Degrees)、第V部 系11.3(Refactoring Is Derived-Geometric)、代数的 Morse 縮約定理(本提案 NT-24)。
- **非自明性**: 定理5.2 は Alexander dual を「repair hitting set」として読むだけで、その代数的複雑度(regularity / projective dimension / 線形分解)と障害コホモロジー次数との定量的関係が無い。本候補は Eagon-Reiner 定理(SR ideal の Alexander dual が線形分解 $\Leftrightarrow$ 元の SR 環が CM)と Terai の公式($\mathrm{pd}=\mathrm{reg}$ of dual)を AAT へ移し、「修復ターゲットの代数的硬さ」と「障害の最高次数」を一つの等式で結ぶ。これは双対性(標準 AG の Alexander / Serre 双対の組合せ影)の AAT 類似であり、G1〜G8 や既存予想(boundary holonomy 等)とは全く別の道具立てである。原則5.3 が「hitting set は repair semantics ではない」と境界を引いたところに、代数的複雑度という測れる構造を初めて与える。
- **CS/SWE帰結**: minimal repair hitting set の「集合としての大きさ」だけでなく、その分解の硬さ($\mathrm{reg}$)を測ることで、「最小修復集合を実装するのに必要な syzygy(相互依存する修復ステップの連鎖)の深さ」が $\mathrm{pd}$(= 必要な分解段数)として数値化できる。CS 読み: $\mathrm{reg}(I^\vee)$ が大きいほど repair hitting set 内の witness を打つ操作が互いに高次に絡む(独立に打てない)。$\mathrm{reg}$ が最小(線形分解)$\Leftrightarrow$ lawful locus が CM $\Leftrightarrow$「修復が段階的に独立して進む良い構造」。これは「どの witness を打つか(hitting set)」に加えて「打つステップがどれだけ絡むか($\mathrm{reg}$)」という、修復計画の本質的並列不能性を初めて測る。移植仮定付き計測接続として、Macaulay2 で Alexander dual + regularity を計算し repair packet に載せる。離散 Morse 縮約を前処理に使う。
- **証明/根拠の見込み**: AAT 仮定で $I_{\mathrm{Ob}}^U$ が SR ideal、$I^\vee$ がその Alexander dual ideal(定理5.2 で構成済み)なので、Terai の公式 $\mathrm{pd}(R/I_\Delta)=\mathrm{reg}(I_\Delta^\vee)$ と Eagon-Reiner 定理($I_\Delta^\vee$ が線形分解 $\Leftrightarrow$ $R/I_\Delta$ が CM; Eagon-Reiner 1998, Herzog-Hibi『Monomial Ideals』8.1.9)がそのまま適用できる。(3) は $\mathrm{pd}$ と Auslander-Buchsbaum / 局所コホモロジー(NT-18)経由で障害最高次数へ接続し、極小自由分解の長さが最高 Tor 次数 = 最高 coherence 障害次数を下から押さえることによる。極小分解の計算は NT-24 の代数的 Morse で実行。AAT の追加仕事は (i) $I^\vee$ の生成元 = repair hitting set の同一視(定理5.2 済み)、(ii) $\mathrm{reg}/\mathrm{pd}$ を第IV部障害次数の reading へ翻訳する regime 固定のみ。証拠: 付録B $I_\Delta=\langle pq,qr\rangle$ の Alexander dual は $I^\vee=\langle q, pr\rangle$(repair hitting sets $\{q\},\{p,r\}$、B.4)で、これは $\mathrm{pd}(R/\langle pq,qr\rangle)=2$ と $\mathrm{reg}(\langle q,pr\rangle)$ の小計算で整合確認できる最小ケースである。
- **関連**: 第VIII部 Alexander dual、B10、NT-20。

### NT-22 Castelnuovo-Mumford レビュー地平定理: regularity が修正波及の打ち止め witness サイズを与える 〔定理候補・証明可能見込み〕

- **主張**: 固定 reading $p$ の square-free witness regime で $I_{\mathrm{Ob}}^U(W)=I_{\Delta_U}(W)\subset R_W=k[E_W]$ とし、Castelnuovo-Mumford regularity を $\mathrm{reg}(I_{\mathrm{Ob}}^U(W)):=\max\{\,j-i\mid \beta_{i,j}(I_{\mathrm{Ob}}^U(W))\neq 0\,\}$($j$ は total degree)で定義する。次が成り立つ。
  - (A) [Terai 公式] $\mathrm{reg}(I_{\mathrm{Ob}}^U(W))=\mathrm{pd}(k[\Delta_U^\vee])=|E_W|-\mathrm{depth}(k[\Delta_U^\vee])$。ここで $\Delta_U^\vee$ は Alexander dual 複体(第VIII部 定理5.2)。
  - (B) [レビュー地平] $m_W:=\mathrm{reg}(I_{\mathrm{Ob}}^U(W))$ とおくと、サイズ $>m_W$ の任意の witness support 結合に新しい「本質的に新規な」minimal syzygy は現れない: すべての高次冗長関係は $m_W$ 以下のサイズの forbidden support と minimal repair hitting set の組合せから線形に生成される。形式的には、$R_W$ の $m_W$-truncation 以降で minimal resolution の差分が線形(linear strand に乗る)。
  - (C) [双対会計] $\mathrm{reg}(I_{\mathrm{Ob}}^U(W))+\mathrm{depth}(k[\Delta_U^\vee])=|E_W|$ が常に成り立ち、repair hitting set 側(Alexander dual)の depth が低いほどレビュー地平が遠い。
  - 相対化: $m_W$ は選ばれた witness regime / $U/J$ に相対化され、未測定 witness を含む拡張 regime では別の $m_{W'}$ になりうる。

### NT-25 破壊的変更の Tor 長下界定理: derived 衝突の最大次数が repair step 数を下から押さえる 〔予想・予想(強)〕

## 5. T5. コホモロジー会計: GRR・Künneth・スペクトル系列

粒度の変更・独立な合成・多 feature 統合という三つの操作は、いずれもコホモロジーの「会計」として捉えられる。粒度を変える集約射に沿っては GRR 型の押し出しが障害の Euler 標数を保存し、変数を共有しない独立な合成では Künneth が障害を因子のテンソル積へ分解し、三つ以上の feature を同時に統合するときは Massey 積とスペクトル系列の退化が高次の干渉を測る。これらは G5(法則干渉の Hilbert 級数恒等式)と G6(集約五項完全列)を、低次の完全列から全次数の保存則・分解則・退化判定へと精密化・拡張するものである。以下の候補は、いずれも選ばれた aggregation family / coefficient / cover / law universe に相対化された数学命題であり、計測に関わる部分は移植仮定付きで述べる。共通して有限 poset regime と体係数の下で有限線形代数に帰着する点が、検算可能性を支えている。

### NT-26 集約 GRR 会計定理: 粒度変更に沿う負債 Euler 標数の押し出し保存 〔定理候補・証明可能見込み〕
- **主張**: 有限 poset site の集約射 π: X_fine → X_coarse(module → package のような粒度変更)と有限階数 abelian obstruction sheaf Ob を固定する。各 sheaf F に対し Euler 標数 χ(F) := Σ_n (−1)^n dim_k H^n(−, F) と、次元の交代和を成分とする負債キャラクター ch(F) := Σ_n (−1)^n [H^n(−, F)] ∈ K_0 を置く。π の高次順像 R^q π_* Ob を係数とするとき、(i) GRR 型会計恒等式 χ_coarse( Σ_q (−1)^q R^q π_* Ob ) = χ_fine(Ob) が成立する(π に沿った負債 Euler 標数の押し出し保存)。(ii) より精密に、各次数で dim H^n(fine, Ob) = Σ_{p+q=n} dim E_∞^{p,q}(Leray)、E_2^{p,q} = H^p(coarse, R^q π_* Ob) であり、両尺度の負債キャラクターの差は Leray 微分の像/核(集約単位内に局在する成分)で完全に勘定される。(iii) cover shape と stalk 次元を保つ refactor は両尺度の χ を同時に保存し、E_∞ の次数分布(尺度安定負債の重み)のみを動かしうる。すべて選ばれた aggregation family / coefficient / cover に相対化される。
- **依拠**: 第IV部 系12.5 Euler Accounting、第IV部 定義14.1 / 定理候補14.2 Leray Five-Term Debt Sequence、第IV部 定義14.3 Scale-Stable Debt、G6 集約五項完全列、G5 法則干渉の Hilbert 級数恒等式、第VIII部 定理4.2 Finite AAT Computability。
- **非自明性**: G6 は Leray 五項完全列(H^1, H^2 の低次のみ)を与えるだけで、全次数を束ねる保存則・指数定理型恒等式には到達していない。本定理は Grothendieck–Riemann–Roch の核心「高次順像の Euler 標数が押し出しで保存される(ch と Todd の整合)」の AAT 類似を立て、全次数の負債 Euler 標数が aggregation で恒等的に保たれること、その差が Leray スペクトル系列の E_∞ で完全に会計されることを主張する。系12.5 の Euler 保存(単一尺度)を二尺度間の押し出し保存へ昇格させ、G5 の Hilbert 級数交代和(law conflict 方向)と相補的な「尺度方向の交代和保存」を与える。これは焼き直しではなく、保存則を尺度変換の関手性として深めるものである。

- **証明/根拠の見込み**: (i) は有限 poset 射の Leray スペクトル系列 E_2^{p,q} = H^p(coarse, R^q π_* Ob) ⇒ H^{p+q}(fine, Ob)(有限なので収束は自明、G6 五項列の全次数版)の Euler 標数が各ページで保存される標準事実による: Σ (−1)^{p+q} dim E_r^{p,q} は r に依らず一定で、E_2 と E_∞ で等しいから Σ_n (−1)^n dim H^n(fine, Ob) = Σ_{p,q} (−1)^{p+q} dim H^p(coarse, R^q π_* Ob)。(ii) は同スペクトル系列の次数別収束。(iii) は系12.5(cover shape と stalk 保存で χ 不変)を π の両側に適用。証明経路は有限線形代数 + 標準スペクトル系列のみで、定理候補14.2 を超える高次項も有限性から E_∞ で止まるため厳密に勘定できる。Macaulay2 / 有限次元計算で検算可能。残る設計: R^q π_* Ob の具体計算手続きを EffCoeff_M に追加すれば即実装できる。

### NT-27 法則干渉の Künneth 公式と衝突消滅: 変数分離なら全次 derived conflict が消える 〔定理候補・証明可能見込み〕
- **主張**: common ambient (X, O_X) 上で、square-free monomial obstruction ideal I_U ⊂ k[E_U]、I_V ⊂ k[E_V] を持つ二つの law universe を取る。witness 変数集合が交わらない(E_U ∩ E_V = ∅)、すなわち law universe が変数分離的に独立であるとき、テンソル分解 R = k[E_U] ⊗_k k[E_V] の下で
  LawConflict_i(U, V) = Tor_i^R(R/I_U^e, R/I_V^e) = 0 (全 i ≥ 1)
が成り立つ(変数分離なら全次の derived conflict が消える = derived-transverse)。より一般に E_U ∩ E_V = S ≠ ∅ のとき、共有変数だけの ambient k[S] 上の Tor で Künneth 型分解
  LawConflict_i(U, V) ≅ ⊕_{a+b=i} Tor_a^{k[S]}( Ī_U, Ī_V ) ⊗ (自由部分)
が成り立ち、derived conflict は共有 witness support S に局在する(共有因子のみが Tor を生む)。すべて選ばれた common ambient / grading / witness family に相対化される。
- **依拠**: 第V部 定義5.1 First Law Conflict Sheaf、第V部 命題5.5 Monomial Conflict Calculation、第V部 例5.6 Shared Witness Factor Conflict、第V部 定理7.3 Derived Transversality Criterion、第V部 定理12.2 Hilbert Series Conflict Identity、第VIII部 定理候補9.2 Flat Base Change Stability for LawConflict。
- **非自明性**: 既存 G5(Hilbert 級数干渉恒等式)は H_{R/I_U}·H_{R/I_V}/H_R = Σ(−1)^i H_{LawConflict_i} という Euler 標数(交代和)レベルの会計に留まり、各次 Tor が個別に消えるか・どこに局在するかは言わない(交代和ゼロでも個々の Tor は非零でありうる)。本候補は変数分離独立性から全次 Tor の消滅(= derived-transverse、定理7.3 の充足十分条件)を立て、共有変数があるときは Tor が共有 support に完全局在するという Künneth 分解を与える。これは Euler 標数より強い「各次の精密会計」であり、例5.6(共有因子 x が Tor_1 ≠ 0 を生む)を一般法則へ昇格する。標準 Künneth(独立な空間の Tor は分解する)の monomial AAT 版で、定理候補9.2(base change)とも整合する独立構造定理である。
- **CS/SWE帰結**: 二つの設計ポリシー(例: layering 規約 U と authority 規約 V)が「同じ witness を共有しないなら、合成しても derived な相互干渉(片方を直すと他方が悪化する隠れた conflict)は一切生じない」ことを証明できる。これは「U-safe ∧ V-safe ⇒ jointly safe」(原則3.2 が一般には否定する命題)が成り立つ十分条件を初めて構造的に与える。逆に共有 witness がある場合、本候補は「干渉はその共有 witness 上だけに局在する」ことを保証するので、ポリシー間の衝突を全コードでなく共有 support に限定して監査できる。実務的には「どのポリシー対が安全に独立合成でき、どの対は共有 witness で衝突するか」のポリシー直交性マトリクスを Tor 計算で自動生成できる(移植仮定: witness 変数の分離 / 共有を ArchMap が判定可能なこと)。
- **証明/根拠の見込み**: 戦略は次の通り。(1) 変数分離 E_U ∩ E_V = ∅ の場合、R = k[E_U] ⊗_k k[E_V] は両因子上 flat で、R/I_U^e と R/I_V^e はそれぞれ片因子からの base change だから、flat base change(定理候補9.2)で Tor_i^R(R/I_U^e, R/I_V^e) ≅ ⊕ Tor_a^{k[E_U]}(R/I_U, k[E_U]) ⊗ Tor_b^{k[E_V]}(k[E_V], R/I_V) = 0 (i ≥ 1)(各因子で片方が自由)。これは標準的代数 Künneth である。(2) 共有 S の場合は k[S] へ base change して相対 Künneth スペクトル系列を取り、free 部分が縮約して共有 Tor へ局在する。(3) 例5.6(I_U = ⟨xy⟩, I_V = ⟨xz⟩, 共有 x)で Tor_1 ≠ 0 が z-class として現れ共有 x support に局在する既存計算が、共有 = 非自明・非共有 = 消滅という本候補の予測を支持する。(4) G5 との整合検算: 変数分離なら H_{R/I_U}·H_{R/I_V}/H_R = H_{R/(I_U+I_V)}(干渉級数 Int = 0)で全 Tor 消滅と一致する。未確立: 一般 monomial(square-free でない)や非 flat ambient での補正項は候補として残る。

### NT-28 アーキテクチャ Künneth 定理: 合成システムの障害は因子の障害のテンソル積で決まる 〔定理候補・証明可能見込み〕
- **主張**: 二つの AAT site X = Site_AAT(S_1, V, U, J_1)、Y = Site_AAT(S_2, V, U, J_2) が有限 poset regime にあり、係数 k が体、obstruction sheaf がそれぞれ Ob_U^X, Ob_U^Y であるとする。積 site X ⊠ Y(context は (W, W′) の積、cover は積 cover、係数は外部テンソル Ob_U^X ⊠ Ob_U^Y)を相対化基底とする。このとき体係数の有限 poset regime において Künneth 同型 H^n(X ⊠ Y, Ob_U^X ⊠ Ob_U^Y) ≅ ⊕_{p+q=n} H^p(X, Ob_U^X) ⊗_k H^q(Y, Ob_U^Y) が成り立つ(Tor 項は体係数ゆえ消える)。系として: (i) 二系統が独立に統合された複合システムの第一隠れ結合容量は dim H^1(X ⊠ Y) = dim H^1(X) + dim H^1(Y)(各系統の容量の単純和)、(ii) X が大域 flat(H^{>0} = 0)なら H^n(X ⊠ Y) ≅ H^n(Y)(flat 因子は複合系の障害を増やさない)、(iii) 二つの flat でない系統の積では H^2(X ⊠ Y) ⊇ H^1(X) ⊗ H^1(Y) という「交差項」障害が必然的に出現する。すべて選ばれた係数体・積 cover・law universe に相対化され、非積 cover(系統間に追加 overlap がある統合)には適用しない。
- **依拠**: 第II部 仮定4.3 Pullback and Overlap / context overlap W_i ×_W W_j、第II部 命題7.2C Finite Poset Computation(有限線形代数 Čech)、第IV部 定義3.1–3.3 Čech Obstruction Complex、第IV部 定理12.2 Topological Debt Capacity(dim H^1 の容量読み)、第VIII部 定理4.2 Finite AAT Computability、個別予想ノート: cover nerve / b_1(N(U)) の容量読み(G2)。
- **非自明性**: 既存 G2 は単一 site の nerve から H^1 容量を読むが、「二つの独立システムを合成したとき障害がどう合成されるか」を扱う定理は存在しない。Künneth は標準代数幾何 / 位相幾何の柱だが、AAT には product site すら明示構成されていない(map に product site なし)。系 (iii) の「二つの非 flat 系統の積では交差項 H^1 ⊗ H^1 が H^2 に必然出現」は、独立に見える二系統の合成が新しい三項整合障害を強制的に生むという非自明な構造主張で、Massey 予想とも整合する。G2 の χ 保存則は単一 site 内の次数間移動だが、本定理は site 間の積に対する乗法的会計則 χ(X ⊠ Y) = χ(X)·χ(Y) を与える点で独立である。
- **CS/SWE帰結**: マイクロサービス系統・モノレポ内の独立サブシステム・別チームが開発した二つのモジュール群を「合成」したときの統合障害容量を、各因子を別々に測った値から予測できる(再解析不要)。具体的には、flat な基盤(H^{>0} = 0)の上に feature 系統を載せても障害は増えない(系 ii)を保証でき、逆に二つの「それぞれは健全だが H^1 ≠ 0」な系統を合成すると、交差 H^2 障害が dim H^1(X)·dim H^1(Y) 次元ぶん必ず生まれる(系 iii)という、合成設計の下界を与える。これは「独立に開発・テストした二系統を結合するとなぜ新種の統合バグが出るか」の代数的説明であり、合成前にコスト(交差項の次元)を見積もる新しい道である(移植仮定: 積 site の構成が ArchMap 側で表現可能であること)。
- **証明/根拠の見込み**: 有限 poset regime では Čech complex C^•(X)、C^•(Y) はともに有限次元 k-cochain complex。積 cover の Čech complex はテンソル積複体 C^•(X) ⊗_k C^•(Y) と擬同型(Eilenberg–Zilber / shuffle 写像の有限版、acyclic cover を仮定)。体係数なので代数的 Künneth 定理(Tor_1 = 0)が直接適用でき H^n(C^•(X) ⊗ C^•(Y)) ≅ ⊕ H^p ⊗ H^q。系 (i)–(iii) は次元計算の即時帰結。χ の乗法性は dim(C^• ⊗ C^•) = dim C^• · dim C^• から従う。支持証拠: 擬円周(H^1 ≅ Z)二つの積 = 擬トーラスで H^1 ≅ Z^2, H^2 ≅ Z という古典トーラスの cohomology に一致する有限例が構成でき、Massey 予想の最小 non-formal 例とも符合する。境界: Eilenberg–Zilber 同型には、積 cover が両因子の acyclic cover の積で acyclic という Leray 型条件が必要で、これを相対化パラメータとして明示する(map の開放端「acyclic cover の十分条件が空白」に対する部分的解答)。
- **関連**: NT-27、G5、第IV部 hidden coupling。

### NT-29 集約射の障害コホモロジー base change 定理: PR-local 計算可能性の転送 〔定理候補・要設計〕
- **主張**: 固定パラメータ p と有限計測 regime のもとで、集約射 π: X_fine → X_coarse(第IV部 定義14.1: module grouping / runtime grouping の有限 site 射)を取り、abelian coefficient Ob を固定する。coarse 側の点(集約単位)w ∈ X_coarse に対し fiber X_w = π^{−1}(w)(その集約単位に属する fine charts と overlaps)と base-change map φ^q_w: (R^q π_* Ob)_w → H^q(X_w, Ob) を考える。次が成り立つ。(i) 半連続性: q = 1 で w ↦ dim_k (R^1 π_* Ob)_w は X_coarse 上半連続。(ii) base-change 同型条件: ある w で φ^q_w が全射ならば近傍の全 w′ で φ^q_{w′} は同型(かつ R^q π_* Ob は w 近傍で局所自由)、すなわち集約単位内部の H^q が coarse 側 stalk として忠実に読める。(iii) cohomology-and-base-change: φ^q_w が全射 iff φ^{q−1}_w が全射、という連結条件のもとで R^q π_* Ob のファイバー値が点ごとの H^q(X_w) と一致する。これは G6(Leray 五項完全列)とは独立で、五項列の項 H^0(coarse, R^1 π_* Ob) が実際に fiberwise H^1 として計算可能になる条件を与える。すべて p / aggregation family / coefficient に相対化される。
- **依拠**: 第IV部 定義14.1 Leray Five-Term Debt Sequence / R^q π_* Ob、第IV部 定義14.3 Scale-Stable Debt、第VIII部 定理候補9.2 Flat Base Change Stability for LawConflict、第VIII部 定義8.1 Cellular sheaf measurement model、付録 Parameter Functoriality (A.2.1) coverage refinement comparison map、拡張ノート G6 集約五項完全列(尺度安定負債)。
- **非自明性**: G6 は Leray 五項完全列で fine の隠れ結合を coarse 可視成分と集約単位内局在成分(R^1 π_*)へ「分配」するが、その R^1 π_* Ob の項が実際に集約単位ごとの H^1 として計算できるか(stalk = fiber cohomology)は保証しない。本候補は Grothendieck の「コホモロジーと基底変換」定理(R^q f_* が点でファイバーコホモロジーを計算する全射条件と局所自由性)の AAT 類似を与え、(1) 集約単位ごとに局所計算した H^1 を貼り合わせれば coarse 側 R^1 π_* が復元できる十分条件、(2) その値が近傍で安定(局所自由)になる条件、を明示する。第VIII部 定理候補9.2(LawConflict の flat base change)は Tor の base change で、本候補は障害コホモロジーの pushforward base change なので対象が異なる。G6 とも「分配 vs ファイバー計算可能性」で役割が異なる。
- **CS/SWE帰結**: (a) 分散検証の核心として、「各集約単位(チーム所有のサブシステム / サービス群)で局所に計算した隠れ結合 H^1 を、再集計するだけで全体の集約レベル隠れ結合 R^1 π_* が正しく得られる」ための十分条件(base-change 全射 + 連結条件)を与える。これにより全コードベースを一度に解析せず、PR が触れた集約単位だけ再計算 → 貼り合わせ、という incremental / 分散な障害計算が健全になる(条件が満たされる限り)。(b) 局所自由性(近傍で dim 一定)は「この集約単位の隠れ結合容量は隣接変更で勝手に変わらない」という安定性を意味し、PR-local な計算結果の再利用可能性を保証する。(c) base-change が破れる集約単位(φ が全射でない w)は「そこだけは局所計算が全体を反映しない = 全体再解析が必要なホットスポット」として特定でき、分散計算の例外箇所を明示できる(移植仮定: aggregation の有限 site 射と abelian coefficient が ArchMap grouping 契約から取れること)。

## 6. T6. 双対性・消滅・K理論・特性類

双対性・消滅・K理論・特性類という四つの古典的道具立ては、AAT の障害コホモロジーに対して互いに補完的な構造を与える。Serre/Poincaré/Verdier 双対は障害類と双対サイクルのあいだに完全対を作り、「H^1 が非零」という存在主張を「それを一周する具体的な統合ループが必ず存在する」という構成的証拠へ昇格させる。消滅定理は nerve の組合せ的な形と overlap の acyclicity から高次障害の不在条件を切り出し、どこまでローカルにレビューすれば大域安全かというレビュー深度の上限を与える。K理論と Euler/Chern 類は、個々の係数層の Euler 標数を debt 同値類のなす群構造へ組織化し、複数 law を同時に整合させる repair の存在可能性を単一の位相障害として読む道を開く。以下の候補群は、いずれも固定された有限 measurement profile に相対化された数学命題として述べられ、計測に降ろす部分には移植仮定を明示する。

### NT-32 AAT 障害 K 群と Euler 標数の準同型性 〔定理候補・証明可能見込み〕

- **主張**: 固定された有限 measurement profile M(finite poset site X_M、U-adequate cover、有限次元係数 Ob_M、EffCoeff_M)に対し、X_M 上の「有限表示 Ob_M-加群層で短完全列について閉じたもの」がなす exact category Coh_M を考え、その Grothendieck 群を K_0(Coh_M) とする。次の二点を主張する。(1) 射 [F] ↦ Σ_n (-1)^n [H^n(U_M, F)] は群準同型 chi_M: K_0(Coh_M) → K_0(Vect_{k_M}) ≅ Z を well-defined に与え、chi_M([F]) = Σ_n (-1)^n dim_{k_M} H^n(U_M, F)。(2) 短完全列 0 → F' → F → F'' → 0 に対し chi_M([F]) = chi_M([F']) + chi_M([F''])(加法性)が成り立つ。相対化: profile M(係数・cover・site)を変えれば一般に別の K_0 と別の chi_M になる。

### NT-33 適格 cover の消滅・比較定理: acyclic cover で Čech が sheaf cohomology を計算する 〔定理候補・証明可能見込み〕

- **主張**: AAT site Site_AAT(S,V,U,J) と obstruction sheaf Ob_U を固定する。cover U = {W_i → W} が U-acyclic、すなわち全ての有限 overlap W_{i_0…i_p} 上で H^q(W_{i_0…i_p}, Ob_U) = 0 (q ≥ 1) であるとする。このとき (A 比較) 自然写像 Ȟ^n(U, Ob_U) → H^n(W, Ob_U) は全 n で同型(Leray/acyclic-cover 定理の AAT 版で、Čech が sheaf cohomology を計算する)。さらに (B 消滅) nerve N(U) の組合せ次元が d なら全 n > d で H^n(W, Ob_U) = 0。とくに N(U) が森(b_1 = 0)かつ acyclic cover なら H^n(W, Ob_U) = 0 (n ≥ 1) で、local lawful sections は(effective torsor regime で)常に大域へ貼り合う。仮定は U-adequate cover、restriction-stable witness ideal、coefficient descent に相対化される。
- **依拠**: 第II部 命題7.2C Finite Poset Computation、第II部 補題7.2A Witness-Closure Cover、第IV部 定理12.4 Local Gluing Sufficiency、第IV部 定理4.1 Obstruction Cohomology、第III部 定義6.2 Obstruction Ideal Sheaf。

## 7. T7. 安定性・モジュライ・降下・スタック

T7 群は、分解の「良し悪し」「合流」「自由度」「合意可能性」という、これまで定性的にしか語れなかった設計判断を、代数幾何の安定性理論で序列化しようとする試みである。GIT/VGIT は分解の安定性を半安定性の数値判定として、付値判定法は version control の confluence と完備性を properness/separatedness として、Kodaira-Spencer はリファクタの一次自由度を接空間の次元として、Galois/gerbe 降下は分割合意の非可換障害を群コホモロジーの消滅として捉え直す。いずれも第VI部の「canonical decomposition は一般に存在しない」という否定的結論を出発点に、その否定性を整理・序列化・局在化する正の対象(安定分解の moduli、マージ極限、refactor 接空間、降下する対称性)へ転じる軸を立てる。すべての主張は選ばれた ArchMap / polarization / law universe / finite regime に相対化された数学命題であり、計測に関わる部分は移植仮定付きで述べる。

## 8. T8. 最適輸送・熱流・スペクトル流

最適輸送理論は、負債を support 上の測度として捉え直すことで、AAT のこれまで静的・代数的だった負債論に動的・双対的な二面性を与える。本テーマは、Kantorovich 双対による review-priority potential、調和質量を輸送コストの下界へ結ぶ比較定理、勾配流・熱流としてのリファクタ収束、そして輸送 Lyapunov による進化軌道の停留を一本の輸送幾何で束ねる。G3 の調和的負債分解が「どれだけ本質的な負債が残るか」を静的に測ったのに対し、ここでは「その負債がどの方向へ・どれだけの距離を運ばれるか」という輸送の言葉でその動的・双対版を記述する。いずれの命題も固定された finite measurement regime に相対化され、計測の橋は移植仮定として明示し、未計測 axis には沈黙する。

### NT-46 負債輸送の Kantorovich 双対と 1-Lipschitz レビュー優先ポテンシャル 〔定理候補・証明可能見込み〕

- **主張**: 固定 finite measurement regime $M$(support グラフ $G_S$、ground 距離 $d_S$、obstruction measure $\Omega_U$)において、repair operation $\mathrm{op}\colon A \to B$ が総質量保存 $|\Omega_U(A)| = |\Omega_U(B)|$ を満たすとき、Kantorovich-Rubinstein 双対が厳密に成立する:
  $$W_1(\Omega_U(A), \Omega_U(B); d_S) = \max\Big\{ \textstyle\sum_{v \in \mathrm{Support}} \varphi(v)\cdot(\Omega_U(A) - \Omega_U(B))(v) \ :\ \varphi\ \text{は}\ d_S\ \text{に関し 1-Lipschitz} \Big\}.$$
  最大化子 $\varphi^*$ を review-priority potential と呼ぶ。(1) $\varphi^*$ は有限 LP の双対解として計算可能で、$\varphi^*(v) - \varphi^*(w) = d_S(v,w)$ を満たす緊張辺 (tight edge) の集合 $T(\varphi^*)$ は質量が実際に流れた transport 経路を含む。(2)【検出規律】全 1-Lipschitz $\varphi$ で $\sum \varphi\cdot(\Omega_U(A)-\Omega_U(B)) = 0$ なら $\Omega_U(A) = \Omega_U(B)$、すなわち measured: 負債は一切移動していない(op は debt について構造的に no-op)。$\varphi^*(v)$ の大きい support $v$ が「負債が最も大きく出入りした review focus」を与える。結論は $M$ に相対化され、未計測 axis の potential は unmeasured として沈黙する。
- **依拠**: 第VIII部 定義10.6 Wasserstein Transfer Cost、定義3.1 Measurement Verdict(5値)、定理候補10.7 Transfer Cost Lower Bound(双対は未定式化)、拡張ノート B9(「有限なので LP で厳密に解ける」とのみ述べ双対定理・potential 読みは不在)、第VII部 定義15.3 U-Detecting Representation Family、個別予想 ACTS Theorem 3 Spectral Review Focus(固有ベクトル版、双対 potential 版は新規)。
- **非自明性**: B9 は「LP で解ける」と一言述べるだけで、(a) 双対が AAT debt measure 上で厳密に成立する条件(総質量保存)を定理化しておらず、(b) 双対解 $\varphi^*$ を review-priority potential として CS 的に読む橋を与えず、(c) tight-edge による transport 経路の特徴づけと no-op 検出規律(双対側からの measured_zero certificate)を全く持たない。Kantorovich 双対は最適輸送の中心定理であり、その AAT 移植は「主問題=どれだけ移ったか」と「双対問題=どの方向に negotiation したか」の二面性を初めて AAT verdict に持ち込む。ACTS Theorem 3 が対称 Laplacian の固有ベクトルで focus を読むのに対し、本候補は非対称な輸送の双対 potential で読み、原理が異なる。
- **CS/SWE帰結**: 「この修復で負債が移ったか/消えたか/単に名前が変わっただけか」に、双対 potential $\varphi^*$ という証拠付きの判定を与える。$\varphi^*$ の大きい support = 「次に review すべき優先箇所」が、ヒューリスティックではなく LP 双対解として算出される(ArchSig の review focus 出力の数学的裏づけ)。tight edge 集合は「負債が実際に通った経路」を可視化し、ArchSig 観測の pressure chain(request-authority → transaction → domain-artifact → AI-surface)を双対 potential の緊張辺として説明できる。全 1-Lipschitz テストで 0 という双対 certificate は「この op は debt について measured no-op」という肯定的・沈黙整合な verdict を与える(移植仮定: ground 距離と obstruction measure を固定した profile 内)。
- **証明/根拠の見込み**: 有限離散最適輸送では $W_1 = \min\text{-cost-flow}$ であり、その LP 双対が Kantorovich-Rubinstein 双対そのもの(強双対定理は有限 LP で無条件、相補性条件が tight-edge を与える)。総質量保存は signed measure $\Omega_U(A)-\Omega_U(B)$ の総和を $0$ に保ち、双対の定数シフト不変性と整合して双対値を finite かつ達成可能にする(保存がないと $W_1$ は $\infty$ または不平衡輸送になり双対形が変わるため、保存が本質的仮定)。tight edge の特徴づけは LP 相補性スラックネス、no-op 検出は「全 1-Lipschitz テストで pairing $0$ $\Rightarrow$ 測度差 $0$」で、$d_S$ が距離ゆえ 1-Lipschitz 関数が点を分離することから従う。証拠として、擬円周 cover 上で $\Omega$ 差 $(1,0)$ に対し $\varphi^*(P)-\varphi^*(Q)=d_S(P,Q)$ を満たす potential を手計算で構成でき、$W_1 = d_S(P,Q)$ と一致する。
- **関連**: B9、G4、NT-47。

### NT-47 輸送下界=調和質量: 本質負債の輸送的不可避性 〔定理候補・証明可能見込み〕

- **主張**: 固定 finite cellular sheaf measurement model $M$(cochain 複体 $(C^n, d_n)$、内積、sheaf Laplacian $L_1$、mismatch 1-cocycle $g$)で、obstruction measure $\Omega_U$ が cochain $g$ の係数を support 上の非負質量へ写す calibration($c^{-1}|g_e| \le \Omega_U\text{-mass}_e \le c|g_e|$、定数 $c$)を固定する。lawful 状態への任意の repair op $A \to A'$ が局所調整(profile 不変な 0-cochain 張り替え)のみで到達されるとき、輸送コストの下界
  $$W_1(\Omega_U(A), \Omega_U(A')) \ \ge\ \kappa_M \cdot \|h(g)\|$$
  が成立する($\|h(g)\|$ は $g$ の調和代表元ノルム = G3 の本質負債質量、$\kappa_M > 0$ は ground 距離・calibration・複体形から定まる定数)。特に $\|h(g)\| > 0$ のとき、いかなる局所調整 repair も $W_1$ で正の輸送コストを払わざるを得ず、調和成分は局所調整では輸送先で消えず別 support へ移送されるのみである。すべて $M$(内積/calibration/ground 距離)に相対化される。
- **依拠**: 拡張ノート G3 調和的負債分解定理(本質修復下界 $\|h(g)\|$)、第VIII部 定理8.5 Finite Hodge Decomposition / 定理8.6 Harmonic Debt Minimality / 系8.7 Essential Repair Lower Bound、定義10.6 Wasserstein Transfer Cost / 定理候補10.7、定義8.3 Distance-to-Flatness Reading、拡張ノート B9 Wasserstein・B3+ harmonic hotspot。
- **非自明性**: G3 / 系8.7 は repair cost が cochain ノルムに $L$-Lipschitz という抽象 cost model 仮定の下で $\|h(g)\|/L$ 下界を出すが、輸送($W_1$)という具体的・計算可能・双対付きの cost に対しては未確立である。本候補は Hodge 理論(調和分解)と最適輸送($W_1$)という二つの異なる強構造を結ぶ比較定理で、harmonic mass という代数的不変量が transport cost という解析的・組合せ的量の下界になることを主張する。これは標準数学でも非自明な calibration を要する(調和性は内積、輸送は ground 距離、両者を $c$ で結ぶ)。G3 の Lipschitz 仮定を、輸送の場合の calibration 不等式へ置き換えて具体化する点が新しい。
- **CS/SWE帰結**: 「局所 patch をいくら重ねても、本質負債質量 $\|h(g)\|$ の分は必ず輸送コストとして現れる」ことを、LP で計算できる $W_1$ の下界として与える。すなわち調和質量(LAPACK で計算)が輸送コスト(LP ソルバで計算)の下限を保証し、spectral と transport という二つの独立な計測パイプラインの整合検算ができる。CS 的には「この負債を局所修正で動かそうとすると最低でも $\kappa_M\cdot\|h(g)\|$ のコストがかかる。それ未満で消えたと報告されたら calibration か coverage に穴がある」という下界付き監査を可能にする(移植仮定: calibration 不等式と ground 距離を固定した cellular sheaf profile 内)。near-miss 検出(spectral gap)と輸送コストを結ぶ実務指標になる。
- **証明/根拠の見込み**: (1) G3(b) より局所調整後の残差 $\min_c \|g - d_0 c\| = \|h(g)\|$。(2) calibration 不等式で cochain ノルムと $\Omega$-mass を比較。(3) 調和代表元 $h(g)$ の符号構造から 1-Lipschitz potential $\varphi_h$ を構成し、Kantorovich 双対(NT-46)に代入して $\sum \varphi_h\cdot(\Omega(A)-\Omega(A')) \ge \kappa_M\|h(g)\|$ を下から評価する。鍵は「調和成分は coboundary(局所調整)で消えないのでその質量は support 間で必ず再配置される $\Rightarrow$ 正の輸送」という直交性と双対の組合せ。有限次元 Hilbert 複体での直交射影(定理8.5)は無条件成立、双対(NT-46)も有限 LP で厳密。擬円周 model で $h(g)$ を明示計算でき($H^1 \cong \mathbb{Z}$ の生成元)、対応する $\Omega$ 差の $W_1$ を LP で解いて下界の成立を数値確認できる。難所は $\kappa_M$ の最適定数(calibration constant と ground 距離の幾何の積)で、これが定理候補に留める要因である。

### NT-49 AAT 熱流リファクタ収束定理: sheaf 熱方程式の指数収束と本質負債への単調減衰 〔定理候補・証明可能見込み〕

- **主張**: 固定有限計測 profile $M$(内積付き cellular sheaf model、$L_1 = $ sheaf Laplacian)と mismatch 1-cochain $g_0 \in C^1$ を相対化基底とする。離散熱流 refactor を勾配流 $dg/dt = -L_1 g$(連続時間)または $g_{t+1} = (I - \eta L_1) g_t$($0 < \eta < 2/\lambda_{\max}(L_1)$ の明示ステップ)で定義する。これは harmonic 成分を保ち coboundary 方向のみを減衰させる勾配降下である。このとき(単調 + 指数収束):
  (i) Lyapunov 量 $\|g_t - h(g_0)\|^2$ は $t$ に沿って単調非増加、
  (ii) $g_t \to h(g_0)$($g_0$ の調和代表元、G3 の本質負債成分)へ収束し、残差は $\|g_t - h(g_0)\| \le e^{-\lambda_1^+(L_1)\, t}\,\|g_0 - h(g_0)\|$(離散版は $(1-\eta\lambda_1^+)^t$)、

  さらに spectral gap $\lambda_1^+(L_1)$ が混合時間 $t_{\mathrm{mix}} = O\big(\tfrac{1}{\lambda_1^+}\log\tfrac{1}{\varepsilon}\big)$ を支配する。結論は profile / 内積 / step $\eta$ に相対化され、未測定 axis の収束は主張しない(計測絡みは移植仮定付き)。
- **依拠**: 第VIII部 定義8.2 Sheaf Laplacian $L_n$、定義8.8 Spectral Gap $\lambda_1^+(L_1)$、定理8.5 / 8.6 Finite Hodge Decomposition / Harmonic Debt Minimality、G3 調和的負債分解定理(本質修復下界)、第IX部 定義6.1 AAT Lyapunov reading / 定理5.3 Finite Dissipation Stopping。
- **非自明性**: 本文には sheaf Laplacian も harmonic debt も spectral gap も定義されているが、これらを結ぶ時間発展(熱流)が完全に欠けている。第IX部の Lyapunov reading は $\Phi_M=\|h(g)\|$ を Lyapunov 候補として挙げるが、熱方程式 $dg/dt=-L_1 g$ という具体的な勾配流もその指数収束率も混合時間との関係も無い。G3 は本質負債の静的な下界(直交射影による min)を与えるが、本定理は「局所調整列を熱流として明示し、それが調和成分へ指数収束し本質質量は熱流不変」という動的な再証明を与え、収束率を spectral gap で定量化する。これは標準のグラフ/sheaf 熱核理論と勾配流の AAT 類似で、B12(力学系)と B11(spectral gap)を熱流という一本の方程式で束ねる新しい大定理核であり、既存の力学定理 G8(停止性のみ)とは収束率・指数評価の点で本質的に異なる。
- **CS/SWE帰結**: 「連続リファクタ(local patch を反復適用する CI loop)を熱流として設計し、その収束速度を spectral gap で予測する」という新しい設計言語を与える。具体的に測れるもの: (1) local-adjustment refactor を反復したとき何ステップで本質負債だけが残るか(混合時間 $t_{\mathrm{mix}} \approx 1/\lambda_1^+$)、(2) spectral gap が小さいモジュール構造は「リファクタが収束しにくい = 技術的負債の解消が遅い」ことの定量予測、(3) 熱流の不動点が G3 の本質負債 $h(g_0)$ と一致するので「どれだけ patch を重ねても残る負債」を動的シミュレーションで可視化。これにより「この CI ポリシーは何 commit で安定するか」「どこを直すと収束が速くなるか(spectral gap を上げる境界変更)」という予測・最適化が初めて可能になる。LAPACK の固有値計算 + 行列冪で実装でき、移植仮定は内積選択と有限 regime。

### NT-50 Koszul 双対による data-flow / control-flow 法則代数の同値定理 〔予想・予想(探索的)〕

- **主張**: architecture geometry $X$ の局所 chart $W$ 上で、第III部の law algebra $\mathcal{O}_X^U(W) = \mathrm{FreeCommAlg}_k(\mathrm{Coord}_X(W))/J_{\mathrm{struct}}(W)$ が square-free monomial(Stanley-Reisner)regime かつ次数付きで Koszul 代数(線形 minimal free resolution を持つ)と仮定する。同一 chart の二つの座標読みを区別する: データ流読み $A_{\mathrm{data}} = \mathcal{O}_X^U(W)$(component が変数、dependency relation が monomial relation、read/write/owns を edge とする)と制御流読み $A_{\mathrm{ctrl}}$(operation/call が変数、control-flow sequencing が relation、calls/handles/sequences を edge とする)。予想: 適切な polarity 付き grading 下で $A_{\mathrm{ctrl}}$ は $A_{\mathrm{data}}$ の Koszul 双対代数 $A_{\mathrm{data}}^!$ と圏同値であり、有界導来圏 $D^b(A_{\mathrm{data}}\text{-mod}) \simeq D^b(A_{\mathrm{ctrl}}\text{-mod})$ が成立する(BGG 型対応)。この同値の下で、(1) data-flow obstruction ideal $I_{\mathrm{Ob}}^{U,\mathrm{data}}$ の Tor 加群 $\mathrm{LawConflict}_i^{\mathrm{data}}$ と control-flow 側の Ext 加群が Koszul 双対対応 $\mathrm{Tor}_i^{A_{\mathrm{data}}}(k,k) \cong \mathrm{Ext}^i_{A_{\mathrm{ctrl}}}(k,k)^*$ で対応し、(2) data-flow 側の $i$ 次 law conflict は control-flow 側の $(\deg - i)$ 次 law conflict と双対的に対応する(次数反転)。すべて固定された Koszul regime と grading polarity に相対化される。
- **依拠**: 第III部 定義4.1 Free Typed Commutative Algebra、定義5.6B Square-Free Witness Regime / 定理5.6C Stanley-Reisner、第V部 定義5.1 First Law Conflict Sheaf(Tor)、定理12.2 Hilbert Series Conflict Identity、第VII部 定義3.3 / 3.5 Graph/Matrix Representation、第VIII部 LawConflict = Tor / Koszul resolutions。
- **非自明性**: AAT には data-flow と control-flow を同一 architecture object の二つの reading とする視点(static relation vs runtime interaction、第I部)はあるが、両者を圏論的双対として結ぶ定理は皆無である。Koszul 双対は「可換代数(data の同時成立)$\leftrightarrow$ 外積/Lie 代数(操作の sequencing)」を交換する標準強定理で、これを data-flow $\dashv$ control-flow に当てるのは新しい。Stanley-Reisner ideal の Koszul 性(Fröberg の定理: SR 環が Koszul $\iff$ flag complex)という具体的足場があり、単なる比喩ではなく検証可能な予想である。G5(Hilbert 級数干渉)は Tor の交代和を測るだけで双対代数の存在は言わない。
- **CS/SWE帰結**: 「同じ設計をデータ流で見るか制御流で見るかは、Koszul 双対で互いに翻訳できる二つの完全な記述である」を数学的に保証する。実務帰結: (1) dependency graph(data)で検出しにくい control-flow 障害が双対側で低次の Tor として現れる、すなわち片方の解析窓で測りにくい障害を双対窓で測れる。(2) data-flow リファクタ(依存整理)の $i$ 次 conflict が control-flow 側で $(\deg - i)$ 次 conflict に移ることを予言し、第V部の repair transfer(cycle repair → semantic boundary 等)に圏論的根拠を与える。(3) static 解析(data)と dynamic trace 解析(control)の相補性(第II部 runtime cover)が双対の二側面として統一される。移植仮定: 抽出器が data-edge と control-edge を分離して ArchMap に出すこと。
- **証明/根拠の見込み**: 戦略 (a) SR 環の Koszul 性判定(Fröberg: $I_\Delta$ が Koszul $\iff$ Stanley-Reisner ideal が 2 次生成 $\iff$ flag complex)を法則 witness regime に適用し data-flow 側を Koszul 代数として固定。(b) Koszul 双対 $A^!$ を quadratic dual として構成、$A^! = $ 外積側 = control-flow sequencing(非可換 operation を可換構造層上の algebra object として読む、原則2.2 と整合)。(c) BGG 対応 $D^b(A)\simeq D^b(A^!)$ を有限次元 Koszul 代数の標準定理として引用、$\mathrm{Tor}_i^A(k,k)\cong(\mathrm{Ext}^i_{A^!}(k,k))^*$ を適用。証拠/特別な場合: 例5.6 の $I_U=\langle xy\rangle, I_V=\langle xz\rangle$(2 次 monomial = flag complex の SR ideal)は Koszul で、その $\mathrm{Tor}_1\neq 0$ が双対側で $\mathrm{Ext}^1$ として現れることを直接計算で確認できる。一般 Koszul 性は予想(全 chart で flag complex regime が成り立つ十分条件の特定が未解決)。
