# 第X部 Semantic Repair Descent と SAGA 比較定理

第III部では、architectural equation system の residual vanishing として lawfulness を定義し、
symbolic violation coordinates から witness ideal と obstruction quotient を生成した。
第IV部では、local lawful sections が大域へ貼り合わない差を cover-relative Čech obstruction cohomology として定式化した。
第V部では、repair を architecture geometry 上の morphism または section の変形として定義した。

残っていた問いは次である。

```text
local repairs may each succeed.
Do they glue to a global repair?
```

第X部の目的は、この問いを、選ばれた有限データの内部で、途切れない定理の連鎖として閉じることである。
repair の貼り合わせ障害を residual の境界所属として読み(descent 定理)、
それを真の商 `Z^1/B^1` の class へ昇格させ(semantic repair additive `H^1`)、
その `H^1` を、architectural equation system が生成する係数を持つ AAT site 上の
cover-relative Čech `H^1` と比較する(SAGA 比較定理)。

中心命題は次である。

```text
repair gluing is descent.
the obstruction is a residual class.
the coefficient is generated by the architectural equation system.
semantic repair H^1 and cover-relative Čech H^1
agree under the selected comparison.
```

## 1. Part9 から Part10 へ

第IX部までで、AAT geometry の静的構造と時間方向の構造が揃った。
第X部は、第III部・第IV部・第V部が開いたまま残していた局所-大域問題を、
一つの部として閉じる。

第X部の対象は、architecture object `A`、その AAT site、architectural equation system `E`、
そして選ばれた有限の chart・overlap・repair・residual データである。
「選ばれた(selected)」は、入力データへの相対化を意味する。存在主張ではない。

第X部を通じて、次の二語を区別する。

```text
supplied:
  入力として与えられ、適合条件で拘束されるデータ。

generated:
  入力から構成され、証明されるもの。
```

第X部の規律は、定理の結論に現れる構造を supplied certificate として受け取らないことである。
係数、restriction、interpretation、descent、比較の各段は、
どこまでが supplied で、どこからが generated かを主張文の中で固定する。

## 2. Semantic Atom と Repair の閉性

repair の貼り合わせを語るには、まず「repair が何に根拠を持つか」を
atom の粒度で読む語彙が要る。

### 定義 2.1 Semantic Atom Projection

semantic atom projection とは、次の組である。

```text
Λ : semantic atom vocabulary(集合)
K : component vocabulary(集合)
π : Λ -> K
```

`K` は component 水準の読みが区別できる粒度、`Λ` はその背後の semantic 粒度である。
`π` の fiber が単射でない点で、同じ component へ射影される相異なる atom(alias)が生じる。

### 定義 2.2 Semantic Residual Atom

選ばれた cover datum を

```text
𝒰 = (有限 chart 族, overlap 遷移データ θ, holonomy support R(𝒰) subset K)
```

とする。
`R(𝒰)` は、overlap 遷移データが非自明な gluing mismatch(第IV部 定義 5.2 の読み)を示す
component の集合であり、選ばれたデータの一部である。

atom `a in Λ` が semantic residual atom であるとは、

```text
π(a) in R(𝒰)
```

が成り立つことである。residual locus を次で書く。

```text
Res(π,𝒰) = π^{-1}(R(𝒰))
```

residual は選ばれた cover datum `𝒰` から定まる生値であり、判定ではない。

### 定義 2.3 Repair Support と Semantic Closure

repair support とは、部分集合 `S subset Λ` である。
`S` が semantically closed であるとは、

```text
Res(π,𝒰) subset S
```

が成り立つことである。
すなわち、すべての semantic residual atom が `S` に入る。
repair support が semantically closed であるとき、その repair を
semantic repair certificate と読む。

### 定義 2.4 Residual Component Coverage と Faithfulness

repair support `S` に対して、次の二つの predicate を定義する。

```text
residual component covered:
  任意の residual atom a に対し、ある s in S で π(s) = π(a)。

residual component faithful:
  residual atom a と s in S が π(s) = π(a) を満たすなら、a in S。
```

coverage は「同じ component の誰かが support されている」ことであり、
faithfulness は「alias ではなく、当の residual atom 自身へ戻れる」ことである。

### 補題 2.5 Closure Decomposition

repair support `S` について、次は同値である。

```text
S is semantically closed
  iff
S is residual component covered
and
S is residual component faithful
```

### 例 2.6 Coverage Without Faithfulness

`Λ = { t, s, o }`、`π(s) = π(o)`(`s` と `o` は同じ component へ射影される)、
`π(t)` は別の component とする。
residual locus が `{ o }` を含み、repair support が

```text
S = { t, s }
```

であるとき、`S` は residual component covered だが faithful ではなく、
したがって semantically closed ではない。

faithfulness は実質的な条件である。
component 水準で覆われていることは、semantic 水準で閉じていることを含意しない。

## 3. 有限 Semantic Repair-Gluing 複体と Descent 定理

### 定義 3.1 Finite Semantic Repair-Gluing Complex

有限 semantic repair-gluing 複体 `𝔎` とは、次のデータの組である。

```text
charts:
  有限 chart 集合(完全列挙付き)。

overlaps:
  chart 対ごとの有限 overlap 集合 O(x,x')。
  overlap の全体は有限リストで完全に列挙され、
  この列挙の完全性はデータの一部である。

C^0:
  local repair primitive の有限集合(完全列挙付き)。
  各 g in C^0 は、各 overlap ω に対して
  左制限 res_L(g,ω) subset Λ と右制限 res_R(g,ω) subset Λ を持つ。

C^1:
  residual repair-gluing cochain の有限集合(完全列挙付き)。
  各 c in C^1 は、各 overlap ω と atom a に対して
  「c が ω 上で a を charge する」predicate ev(c,ω,a) を定める。

δ^0 : C^0 -> C^1:
  restriction-difference rule を満たす微分。
  すべての g, ω, a について
  ev(δ^0 g, ω, a)
    iff
  res_L(g,ω)(a) と res_R(g,ω)(a) のちょうど一方が成り立つ。

r in C^1:
  選ばれた residual(supplied データ)。

semantic 側:
  semantic atom projection (Λ, K, π)、cover datum 𝒰、
  support 割当 supp : C^0 -> P(Λ)。
```

`g in C^0` は「chart ごとの local repair の族を、overlap への読みを通じて記録した 0-cochain」であり、
`δ^0 g` は「大域候補 `g` が overlap 上でどれだけ食い違うか」である。
`δ^0` の restriction-difference rule は supplied データに課された条件であり、導出されたものではない。

cover datum `𝒰` は、複体の charts / overlaps とは別に選ばれるデータである。
residual atom の読み(定義 2.2)は `𝒰` から定まり、
residual cochain `r` は複体の overlap の上のデータである。

### 定義 3.2 境界所属と Global Semantic Repair Coherence

複体 `𝔎` に対して、次を定義する。

```text
B^1 = δ^0(C^0)

obstruction vanishes:
  r in B^1

obstruction is nonzero:
  r not in B^1
```

global semantic repair coherence を次で定義する。

```text
GlobalCoherent(𝔎)
  iff
exists g in C^0,
  δ^0 g = r
  and
  supp(g) is semantically closed
```

すなわち、residual を境界として消す primitive が存在し、
しかもそれが形式解ではなく、定義 2.3 の意味での semantic repair certificate であることを要求する。

この節の障害の消滅は境界所属である。
商としての class と `H^1` の語は、§4 の構成にのみ使う。

### 定義 3.3 Semantic Faithfulness Hypothesis

複体 `𝔎` が semantic faithfulness hypothesis を満たすとは、
`δ^0 g = r` なるすべての `g in C^0` について、`supp(g)` が
residual component covered かつ residual component faithful であることをいう。

### 定理 3.4 Finite Semantic Repair-Gluing Descent [Certified bounded inference]

`𝔎` を有限 semantic repair-gluing 複体とする。

```text
(i) 必要性(無条件):
  GlobalCoherent(𝔎)
    =>
  r in B^1

  対偶として、
  r not in B^1
    =>
  not GlobalCoherent(𝔎)

(ii) 十分性(semantic faithfulness hypothesis の下で):
  r in B^1
    =>
  GlobalCoherent(𝔎)

(iii) descent 同値(semantic faithfulness hypothesis の下で):
  GlobalCoherent(𝔎)
    iff
  r in B^1
```

### 証明の読み

(i) は、global coherence の定義に境界方程式 `δ^0 g = r` が含まれることから直ちに従う。
証明の実体は定義の設計にある。

(ii) が定理の重心である。`r = δ^0 g` なる `g` を取ると、
semantic faithfulness hypothesis が `supp(g)` の coverage と faithfulness を与え、
Closure Decomposition(補題 2.5)により `supp(g)` は semantically closed になる。
したがって `g` は形式解ではなく semantic repair certificate である。

難所は cohomology 代数ではない。
形式解と semantic repair の同一視、すなわち faithfulness にある。

### 定理 3.5 Complete-Support Faithfulness

例 2.6 と同じ、選ばれた semantic atom projection と cover datum の上で、
すべての primitive に complete support

```text
supp(g) = Λ
```

を割り当てた有限複体の族を考える。
この族に属する任意の `𝔎` について、semantic faithfulness hypothesis は定理として成立し、
仮定として供給する必要がない。
したがって族の内部では、無条件に

```text
GlobalCoherent(𝔎)
  iff
r in B^1
```

である。

complete support の閉性は Closure Decomposition により coverage と faithfulness へ分解されるから、
これは仮説が空虚でないことの正例である。
faithfulness 条件の実質は、例 2.6 の負例が担う。

### 例 3.6 Component Coverage Is Not Closure

単一 chart・単一 overlap・`C^1` が二元の複体で、
support を例 2.6 の `S = { t, s }`(`o` を記録しない)とし、
residual `r` が atom `o` を charge するものを取る。

このとき、

```text
r not in B^1
not GlobalCoherent(𝔎)
```

であり、同時に component 水準の被覆と、可視の局所 repair の整合はすべて成立する。

component 水準の合格は境界所属を与えない。
semantic residual の障害は、`π` の fiber の中、すなわち alias の差に残る。

## 4. Semantic Repair Additive H^1

§3 の境界所属を、真の商の class へ昇格させる。

### 定義 4.1 Finite Repair Cover と Cover Nerve

有限 repair cover とは、次のデータの組である。

```text
cover charts:
  有限 chart 族(完全列挙付き)。

pairwise overlaps:
  chart 対ごとの overlap 成分(完全列挙付き)。

triple overlaps:
  chart 三つ組ごとの triple overlap 成分(完全列挙付き)と、
  各 triple overlap の三つの境界辺 ∂01, ∂12, ∂02。
```

この cover は cover nerve(第IV部 定義 12.1 の 2-骨格版)を定める。
頂点は chart、辺は overlap 成分、面は triple overlap 成分とその三辺である。

### 定義 4.2 Čech 型係数データ

選ばれた有限 repair cover 上の Čech 型係数データとは、次の組である。

```text
C^0, C^1, C^2(C^0, C^1 には選ばれた列挙を付す。完全性は要求しない)
0_1 in C^1, 0_2 in C^2
δ^0 : C^0 -> C^1
δ^1 : C^1 -> C^2

compatibility:
  δ^1 ∘ δ^0 = 0_2(すべての primitive で)
  δ^1(0_1) = 0_2
  選ばれた residual r in C^1 は δ^1(r) = 0_2 を満たす(r は 1-cocycle)。
```

`C^0` は local repair primitive、`C^1` は overlap 上の residual cochain、
`C^2` は triple overlap 上の整合性の受け皿である。
`δ^1 ∘ δ^0 = 0_2` が、nerve の面関係の複体としての表現である。

`δ^0`, `δ^1` は cover の幾何から導出されるのではなく、
選ばれた cover に付随して供給されるデータであり、
上の compatibility 条件がその適合性のすべてである。

### 定義 4.3 Additive Regime

Čech 型係数データが additive regime にあるとは、
`C^0`, `C^1` が可換群であり、`0_1 = 0` であり、`δ^0` が群準同型であることをいう。

### 定義 4.4 Semantic Repair Additive H^1

additive regime の Čech 型係数データに対して、次を定義する。

```text
Z^1 = { c in C^1 | δ^1 c = 0_2 }
B^1 = δ^0(C^0)

c ~ c'
  iff
c - c' in B^1

H^1 = Z^1 / ~
```

同値関係の反射・対称・推移は、`δ^0` の準同型性から証明される。
剰余類 `[c]` は well-defined である。

residual class を `[r]`、zero class を `[0]` と書き、

```text
H^1-zero
  iff
[r] = [0]
```

と定義する。

### 補題 4.5 Quotient Zero Reading

```text
[r] = [0]
  iff
r in B^1
```

商の等式と境界所属は一致する。

### 定義 4.6 Boundary-Relation Faithfulness Data

closure predicate `P subset C^0` を供給する。
`P(g)` は「`g` は semantic repair certificate である」と読む。
§3 の複体では `P(g)` は `supp(g)` の semantic closure である。

global semantic repair coherence を次で定義する。

```text
GlobalCoherent
  iff
exists g in C^0,
  δ^0 g = r
  and
  P(g)
```

boundary-relation faithfulness data とは、次の組である。

```text
zero primitive:
  z in C^0 with δ^0 z = 0_1

residual support predicate:
  Q subset C^1 with Q(r)

faithfulness law:
  Q(δ^0 g) => P(g)
```

この形は、cocycle ごとに primitive を割り当てる大域選択子を要求しない。
additive regime では zero primitive として `z = 0` が取れる。
faithfulness data は、additive 性を仮定しない形で固定する。

### 定義 4.7 True Sheaf Certificate

AAT site `S`(第II部 定義 8.1)、presheaf `F`、
および `S` の上で別途選ばれた cover に対して、
true sheaf certificate とは次の組である。

```text
cover membership:
  選ばれた cover が生成する篩(sieve: base への射のうち、
  cover のいずれかの chart を経由するものの族)が
  S の topology に属する。

global sheaf condition:
  F が S の topology のすべての cover に対して
  sheaf 条件(第II部 定義 10.1)を満たす。
```

repair cover(定義 4.1)とこの cover の対応は、定義 6.3 の incidence data に委ねる。

### 定理 4.8 True Sheaf H^1 Semantic Repair-Gluing [Certified bounded inference]

選ばれた有限 repair cover、その上の Čech 型係数データ(additive regime)、
boundary-relation faithfulness data、true sheaf certificate を固定する。
このとき、次がすべて成立する。

```text
1. cover sheaf condition and descent:
  F は選ばれた cover に対する sheaf 条件を満たし、
  その cover 上の任意の gluing data に対して一意の大域 section が存在する。
  この cover ごとの条件は per-cover の仮定ではなく、
  global sheaf condition と cover membership から導出される。

2. cocycle:
  r と任意の δ^0-境界は 1-cocycle である。

3. descent equivalence:
  GlobalCoherent
    iff
  [r] = [0] in H^1 = Z^1 / B^1

4. quotient reading:
  [r] = [0]
    iff
  r in B^1

5. higher layers:
  torsor 層(第IV部 原則 2.3)、higher coherence(第IV部 意味 10.2)、
  stack 的構造(第VI部)に対応する各上位条件は、
  additive regime の構成から自明に成立する。外部仮定として供給されない。
```

### 証明の読み

十分性([r] = [0] ⟹ GlobalCoherent)は、商の等式から `r in B^1` を得て、
境界 primitive `g` を取り、`Q(r)` と faithfulness law から `P(g)` を得る。
必要性は定義の設計から従う。

結論 1 と結論 3 は同じ package の並列の結論である。
3 は 1 から導かれるのではなく、faithfulness data と additive regime から導かれる。

この `H^1` は選ばれた cover に相対的な Čech 型の商である。
`H^n(X, ·)` と書くための追加条件は、第IV部 定義 3.3 の規律に従う。

### 命題 4.9 Finite Complex Shadow

§3 の有限複体 `𝔎` と本節の係数データの間に、
`δ^0` と residual を保つ `C^0`・`C^1` の全単射が与えられれば、

```text
[r] = [0] (本節)
  iff
r in B^1(𝔎)
```

である。§3 の境界所属は、本節の `H^1` 零 class の選ばれた有限 shadow である。

### 命題 4.10 Refinement Pullback(片方向)

粗い cover から細かい cover への refinement data
(chart / overlap / triple の写像と、`δ^0`・residual に自然な引き戻し)が与えられれば、
粗い側の境界所属と `H^1`-zero は細かい側へ引き継がれる。
逆方向は、定理 8.5 の反例により、追加データなしには成立しない。

## 5. 係数の生成 — Architectural Equation System

§4 の係数データは supplied だった。
本節では、係数そのものを architectural equation system から生成する。

第I部の architectural equation system `E` と、第III部 定義11.3の displayed equation source を固定する。
symbolic violation coordinates `nu` から witness ideals と obstruction ideal が生成され、
object-dependent residual `epsilon` から displayed defect が定義される。第III部 定理11.4により、
context-wise obstruction quotient presheaf

```text
Q_E(W) = O_E(W) / I_Ob^E(W)
```

が生成され、その restriction、消滅、忠実性、restriction evaluator が定理として従う。
係数は supplied certificate ではない。witness ideal の語彙から構成され、証明される。

### 定義 5.1 Architectural Equation Grounded Surface

architectural equation grounded surface とは、次の選ばれたデータの組である。

```text
full AAT site package:
  architecture object A、equation system E、signature Sig、coverage requirements R、
  context-overlap package Ov : ContextOverlapPullback(ArchCtx(A))、
  generated topology J_{E,R,Ov}。

finite poset atom/equation cover geometry:
  meet を持つ有限 context poset、有限添字の admissible cover 𝒰、
  𝒰 の chart / pullback-overlap から生成される有限 Čech nerve、
  cover の (E,R,Ov)-adequate 性(第II部 定義 7.2)。
  第II部 定義 7.2B の finite poset AAT site regime で読む。

skeleton:
  0-単体ごとに選ばれた required equation index と support Atom。

displayed equation source:
  cover 添字ごとの local context、architecture object、required equation index、
  support Atom。displayed defect は epsilon から定義する(第III部 定義 11.3)。

incidence bridge:
  選ばれた repair cover(定義 4.1)と、その chart / pairwise / triple overlap を
  𝒰 の Čech nerve の 0 / 1 / 2-単体へ、face と restriction を保って同定する
  provenance data(定義 6.3)。

global quotient sheaf condition:
  生成された商係数 presheaf が、full AAT site の topology J_{E,R,Ov} に対して
  sheaf 条件を満たすこと。
```

一般には global quotient sheaf condition は仮定である。
context がただ一つの有限モデル instance の上では、
これは定理として成立し、仮定として供給する必要がない(例 9.1)。

### 意味 5.2 Coefficient Is Generated

architectural equation grounded surface の上では、次が成り立つ。

```text
coefficient:
  nu が生成する witness ideals の商として構成される。

restriction:
  ideal の制限互換から誘導され、functor 法則は証明される。

interpretation:
  epsilon から選ばれた displayed residual の商 class として生成される。

vanishing:
  equation fulfillment から residual zero、ideal membership、quotient zero の順に導く。
```

## 6. Coverage-Generated AAT Site と Čech 橋

§4 の repair cover と、第II部の AAT site・第IV部の Čech 装置を接続する。

### 定理 6.1 Coverage Generation Bridge

選ばれた finite AAT site `Site_AAT(A,E,Sig,R,Ov)` の poset regime
(第II部 定義 7.2B)において、admissible coverage family が生成する篩は
`J_{E,R,Ov}` の cover である。
また、AAT site の topology は、その coverage 要求から生成される位相と一致する。
これは第II部 定義 7.1 の生成による定義的一致である。

atom support と required equation-coordinate coverage を要求する admissible cover から生成された
`J_{E,R,Ov}` が、以下のすべての descent と sheaf 条件の土台である。

### 定義 6.2 Selected Simplicial Čech Complex

cover-relative Čech complex を、選ばれた単体データの上で読む一般形を固定する。
selected simplicial Čech complex とは、次の組である。

```text
base and charts:
  base context と、chart の族(0-単体)。

selected simplices:
  1-単体の族(各 1-単体に source / target の面)、
  2-単体の族(各 2-単体に三つの境界辺 ∂01, ∂12, ∂02)。
  各次数の単体は選ばれたデータである。

cochains and differential:
  C^n は次数 n の単体の上の係数値関数の集合。
  微分は面への制限の交代和。
```

第IV部 定義 3.1〜3.3 の cover は、単体を chart 添字の tuple そのものに取り、
overlap を pullback で読む特別な場合である。
grounded surface ではこの chart-indexed 特別形を使い、単体は実際の cover と `Ov` の pullback overlap から
生成する。任意に選んだ simplicial set は grounded surface の cover nerve と同一視しない。

### 定義 6.3 Repair Cover Incidence Bridge

§4 の有限 repair cover から、grounded surface の cover nerve への橋を、次数 0〜2 の全単射と
face / restriction の可換性として与える。

```text
chart <-> 0-単体
nonempty pullback overlap <-> 1-単体
nonempty triple pullback overlap <-> 2-単体
face maps and coefficient restrictions commute with these identifications
```

### 定理 6.4 Incidence Is Data

この incidence data は、単体データだけからは一様に構成できない。
0-単体の族が空でなく、chart 添字集合が空である selected simplicial cover が反例を与える。

逆に、0-単体が chart 添字そのものである chart-indexed cover からは、
必要な incidence が定義から構成される。

対応関係は仮定でも省略でもなく、データである。

### 定理 6.5 Cover Nerve Provenance

§4 の cover nerve の辺と面は、選ばれた pairwise / triple overlap 成分そのものであり、
面の境界辺は typed に一致する。
nerve は overlap データの再記述であって、別の対象ではない。

### 定理 6.6 Descent Is Derived

presheaf `F` が AAT sheaf 条件を満たし、選ばれた cover の篩が topology に属するなら、
その cover に対する sheaf 条件、descent、
gluing data に対する大域 section の一意存在が、定理として従う。

sheaf 条件と cover membership の二つの入力から、descent は導出される。
独立の descent certificate は要求されない。

## 7. SAGA 比較定理

### 定義 7.1 H^1 Comparison Data

semantic 側の Čech 型係数データ(§4、additive regime)と、
cover-relative Čech complex(定義 6.2 の selected simplicial 形)の間の
`H^1` comparison data とは、次数 0, 1, 2 の両方向写像の組で、次を満たすものである。

```text
degree 1:
  両方向写像は互いに逆であり、差を保つ。

degree 2:
  zero を保つ。

compatibility:
  次数 0 -> 1 と 1 -> 2 の微分と、両方向で可換。
```

comparison data は、商の等式・zero class・大域整合・descent の結論を格納しない。

### 定理 7.2 SAGA 比較定理 [Certified bounded inference]

任意の semantic 係数データ(additive regime)、cover-relative Čech complex、
`H^1` comparison data に対して、次がすべて成立する。

```text
1. cocycle 対応:
  comparison data は semantic 1-cocycle と
  Čech 側の次数 1 cocycle を両方向に写す。

2. coboundary 対応:
  二つの semantic cocycle が同じ class を持つことは、
  それらの像が Čech 側で coboundary 関係にあることと同値である。

3. H^1 比較同値:
  商に誘導される写像は well-defined で互いに逆であり、

  semantic repair additive H^1
    ≅
  H^1(𝒰, coefficient)

  が選ばれた comparison の下で成り立つ。

4. zero-predicate equivalence:
  semantic 側の residual class が零であることと、
  Čech 側の residual の像の class が零であることは同値である。
```

この一般形に有限性の仮定は不要である。
有限性は、grounded surface(定義 5.1)と instance(§9)の側に属する。

逆写像の well-definedness と両側逆性は、
供給されるデータの成分ではなく、商水準で証明される定理である。

### 定理 7.3 Grounded Global Gluing [Certified bounded inference]

定理 7.2 の設定に加えて、boundary-relation faithfulness data(定義 4.6)、
true sheaf certificate(定義 4.7)、gluing data が与えられれば、

```text
global semantic repair coherence
  iff
cover-relative residual H^1 zero
```

が成り立つ。
定理 4.8 の descent 同値は、この形で、
第II部の site・sheaf・descent の地盤の上の定理として再読される。

### 定理 7.4 Nonzero Class Transfer

zero-predicate equivalence の対偶として、
一方の側の class が非零ならば、他方の側の class も非零である。
この転送は、例 9.2 の circle instance の上で、
両側が証明可能に非零な class に対して実際に発火する。

### 定理 7.5 Generated SAGA Input Realization [Certified bounded inference]

architectural equation grounded surface(定義 5.1)から、選択済み入力、生成物、導出結果を区別した
次の幾何側入力面が得られる。

```text
selected inputs:
  full AAT site package、(E,R,Ov)-adequate cover、
  global quotient sheaf condition(定義 5.1)。

generated objects:
  topology J_{E,R,Ov} と displayed cover family の生成篩(定理 6.1)、
  実際の cover nerve 上の Čech complex、
  商係数(第III部 定理 11.4)、
  epsilon から得る displayed residual degree-zero cochain。

derived result:
  selected cover に対する descent(定理 6.6)。
```

これらの構成は displayed equation fulfillment を仮定しない。fulfillment が成立する場合に限り、
さらに次が従う。

```text
displayed residual is pointwise zero.
the generated degree-zero cochain is zero.
the restriction evaluator preserves that zero(第III部 定理 11.4).
```

grounded surface だけから semantic complex、`H^1` comparison data、comparison equivalence、
semantic / additive `H^1` zero は従わない。semantic 係数データと定義7.1の comparison data が
構成または明示されたとき、定理7.2が商水準の比較同値を証明する。さらに定理7.3の
faithfulness / sheaf / gluing data があるとき、global gluing comparison が従う。

### 原則 7.6 SAGA Is Not AAT-GAGA

```text
AAT-GAGA(第VIII部 定理 12.3):
  constructed obstruction data と analytic / combinatorial readings を、
  finite measurement profile の中で束ねる比較 interface。

SAGA(本部 定理 7.2・定理 7.3):
  semantic repair additive H^1 と、AAT site 上の
  cover-relative Čech H^1 の比較定理。
  両辺はどちらも代数的 cohomology である。
```

SAGA は Sémantique Architecturale, Géométrie Algébrique と読む。
SAGA の名は、代数幾何における比較定理の伝統(GAGA)への言及であり、
Serre の定理の適用でも一般化でもない。

## 8. Equation 寄与の所在

equation fulfillment が、この cohomology のどこを動かすかを定理として確定する。

### 定理 8.1 Degree-Zero Equation Contribution [Certified bounded inference]

architectural equation grounded surface の上で displayed equation fulfillment が成立するならば、
生成された 0-cochain は各点で零である。

equation fulfillment が Čech 複体に到達する地点は、正確に cochain 次数 0 であり、
その restriction と coboundary も零になる。これは一般の cover-relative `H^1` の消滅を主張しない。

### 定理 8.2 Equation-Fulfillment Independence of the Grounded Route

定理 7.5 の full AAT site と adequate cover は fulfillment と独立に選択され、topology、cover nerve、
商係数、displayed residual degree-zero cochain、生成篩、descent は fulfillment を仮定せずに得られる。
semantic primitive、semantic complex、comparison data は
この生成物には含まれず、定理7.2を適用する際に別途構成または明示する。

### 意味 8.3 Where Equation Semantics Lives

定理 8.1 と定理 8.2 により、equation fulfillment の寄与の所在が上下から確定する。

```text
law satisfaction acts at H^0: vanishing.
nontrivial H^1 content comes from the geometry of the cover.
```

現行の生成係数では、lawful な reading は商で零になる。
相異なる非零の lawful reading が非自明に貼り合う係数
(第IV部 定義 2.4 の conormal 係数 `I_U/I_U^2` は自然な候補である)を
どう読むかが、本文の残る問いである。

### 定理 8.4 Full Sheaf Comparison Is Data

cover-relative Čech `H^1` から sheaf cohomology への比較は、
明示的な比較データが与えられた場合に限り成立する。

無条件比較は存在しない。
非零な cover-relative `H^1` を持つ有限複体と、
比較の行き先を持たない係数選択の組が存在し、
その上ではいかなる比較写像も構成できない。
比較可能性は、明示的な比較データの存在そのものに相対化される。

### 定理 8.5 Refinement Naturality Is Data

cover refinement に沿う `H^1` の自然性は、
明示的な refinement 比較データ(命題 4.10)の下でのみ成立する。

無条件版は成立しない。
coarse 側の residual が零 cochain であり、fine 側の residual が coboundary でない有限例が存在し、
その上では residual と zero を保つ引き戻しは存在しない。

比較可能性はデータである。これが SAGA 比較定理の外周である。

## 9. 有限 Instance

### 例 9.1 Lawful Firing Instance

第I部・第II部の語彙で構成した、context がただ一つの有限モデルを取る。Atom universe と required
equation index を一つずつ取り、observable ring を `Z`、violation coordinate を `2`、displayed
equation residual を `0` とする。signature は自明な axis、`R` はこの Atom と equation coordinate、
`Ov` は identity pullback を指定し、identity cover を取る。このデータは architectural equation
grounded surface(定義 5.1)を満たし、生成位相は自明、商係数は `Z/(2)` である。
したがって定理 7.5 の結論が発火する。

この site では、admissible cover family が非空の patch を強制し、
単元 poset の上では生成篩の任意の引き戻しが最大篩になる。
したがって任意の presheaf が sheaf 条件を満たし、
特に生成された商係数の global quotient sheaf condition は定理として成立し、
仮定として供給する必要がない。

この例は lawful class の上にあり、defect class は零である。
これは定理 8.1 の帰結であって、欠陥ではない。

### 例 9.2 Circle Nerve Nonzero Class

例9.1とは別に、base context `W` を四つの chart `W_0,W_1,W_2,W_3` で覆う finite AAT site を取る。
非空の pairwise pullback overlap は

```text
W_{01}, W_{12}, W_{23}, W_{30}
```

だけで、non-adjacent overlap と triple overlap は空 context とする。topology `J_{E,R,Ov}` はこの cover と
その pullback から生成する。この `(E,R,Ov)`-adequate cover の
実際の Čech nerve は4-cycleであり、次数2以上の nondegenerate simplexを持たない。
係数を各 nonempty context 上で一定な equation-system 商

```text
Q = Z/(2) ≅ F_2
```

とする(witness ideal は `(2) subset Z`)。
residual は辺 `W_{01}` で `[1]`、他の三辺で `0` を取る 1-cochain である。

```text
r は 1-cocycle である:
  次数 2 の単体が存在しないため、cocycle 条件は自明に成り立つ。

r は coboundary ではない:
  4-cycle 上の任意の coboundary の辺値の総和は 0 だが、r の総和は [1] である。
  非零性は equation-system 商の算術的事実 1 not in (2) に帰着する。
```

この複体を担体とする semantic 係数データと、
定義的に等しい担体上の恒等 comparison の下で、
semantic additive `H^1` の residual class と cover-relative `H^1` の class は共に非零であり、
zero-predicate equivalence が非零性を双方向に転送する(定理 7.4)。

semantic 側は同じ四つの chart、四つの overlap、同じ differential から semantic additive complex を
構成する。この具体例では次数0〜2の恒等写像が定義7.1の comparison data を与える。
詳細な有限計算は付録 B.9 に置く。

例 9.1 と例 9.2 は別の witness である。
lawful な例では class は必然的に零であり(定理 8.1)、
非零 class の転送は circle instance が担う。

## 10. Part10 の結論

第X部では、selected input、generated object、derived result を混同しない次の依存図を確定した。

```text
selected core input:
  Atom universe At、Atom axiom system S、core reading r : CoreRead(At)
  with equation reading r.R_eq and signature reading r.R_sig
    -> architecture object A_r、equation system E_r、signature Sig_r(第I部)

selected coverage input:
  E_r、Sig_r、R_r : CoverageRequirements(A_r,E_r,Sig_r)、
  Ov_r : ContextOverlapPullback(ArchCtx(A_r)) realizing R_r
    -> generated topology J_{E_r,R_r,Ov_r}(第II部 定義 7.1、本部 定理 6.1)

equation-generated algebra:
  nu of E_r
    -> witness ideals and obstruction ideal(第III部)
    -> quotient coefficient O_{E_r} / I_Ob^{E_r}(第III部 定理 11.4)
  epsilon of E_r + displayed equation source
    -> displayed residual interpretation(第III部 定義 11.3・定理 11.4)

selected geometric input:
  (E_r,R_r,Ov_r)-adequate cover、
  global quotient sheaf condition for O_{E_r} / I_Ob^{E_r}、
  repair-cover incidence bridge

generated / derived:
  adequate cover + Ov_r
    -> actual pullback nerve(本部 定理 6.5)
  actual pullback nerve + quotient coefficient
    -> selected cover-relative Čech complex
       (本部 定義 6.2。一般構成は第IV部 定義 3.1〜3.3)
  repair-cover incidence bridge
    -> identifies the semantic cover with the actual pullback nerve in degrees 0 to 2(定義 6.3)
  global quotient sheaf condition + cover membership
    -> descent from sheaf condition and cover membership(本部 定理 6.6)

selected semantic comparison input:
  semantic coefficient data、boundary-relation faithfulness data、
  H^1 comparison data、true sheaf certificate、gluing data

derived:
  semantic coefficient data with residual 1-cocycle
    -> semantic repair H^1 class in Z^1 / B^1(定義 4.4)
  semantic coefficient data + boundary-relation faithfulness data
    + true sheaf certificate + gluing data
    -> global coherence iff residual class is zero(定理 4.8)
  semantic coefficient data + H^1 comparison data
    -> semantic repair H^1 and cover-relative Čech H^1 comparison(定理 7.2)
  faithfulness + true sheaf certificate + gluing data
    -> grounded global gluing comparison(定理 7.3)
```

`E_r` は Atom axioms 単独から生成されるのではなく、core reading `r` の material input `R_eq` から構成される。
同様に `R_r`、`Ov_r`、adequate cover、global quotient sheaf condition、semantic coefficient data、comparison data は
図に明記した selected input である。生成されるのは、それらの入力から矢印の右側に置いた topology、ideal、
quotient、displayed residual、cover nerve 上の complex と descent である。comparison equivalence は定義7.1の
comparison data から商水準で証明し、`H^1` zero を grounded surface の入力に格納しない。

第X部の核心は次である。

```text
repair gluing is descent.
the obstruction is a residual class.
the coefficient and displayed interpretation are generated by the equation system.
comparison data is data, and the comparison is a theorem.
```

これにより、AAT は次を扱える。

```text
locally repaired
but
the residual class remains
```

そしてこの現象は、選ばれた有限データの内部で、`(At,S,r)`、`r.R_eq` から得た `E_r`、
full coverage package、adequate cover、global quotient sheaf condition、semantic coefficient data、
comparison data の明示的な依存図として接地される。

equation fulfillment は `H^0` で消滅として働き、`H^1` の非自明な内容は cover の幾何から来る(意味 8.3)。
相異なる lawful reading が非自明に貼り合う係数の読みは、本文の残る問いである。

第X部の主張は、選ばれた chart、overlap、residual、係数、comparison data に相対化される。
第X部の結論は、固定された semantic repair geometry の内部でだけ読む。
