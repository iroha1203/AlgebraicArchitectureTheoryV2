Algebraic Architecture Theory / Software Field Theory

Software architecture is a geometry.

This site is the public home for AAT, SFT, and ArchSig: a research program that treats a codebase as a geometric object — atoms generate it, laws cut it out, and cohomology reads what holds locally but fails to glue globally — and makes that reading computable.
