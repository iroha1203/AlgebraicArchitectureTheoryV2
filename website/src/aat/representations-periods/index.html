How to read the claims

AAT states its claims relative to a fixed atom vocabulary, law universe, coverage topology, coefficient sheaf, and selected representation family. Three claim grades are kept distinct throughout the monograph.

Formal theorem
A mathematical proposition proved from the axioms, definitions, and stated assumptions.
Certified bounded inference
A relative inference that holds under explicitly stated coverage, exactness, or witness-completeness assumptions. It is never an unconditional absolute claim.
Analytic reading
A way of reading a constructed geometric object through graphs, signatures, periods, or metrics.

Part VII is the chapter where the analytic reading takes the leading role. Most of its statements introduce readings — representations, periods, metrics, masses, and costs — of the geometry built in Parts I through VI, and the theorems of this part are certified bounded inferences that say exactly when a reading preserves or reflects structure. Theorem labels on this page follow the canonical text. A label carries a Lean status badge only when the AG Lean index has a corresponding ledger row; a label without a badge is a mathematical statement in the canonical text and is not read as a current Lean proved claim.

2. Representation as functor

Definition VII.2.1 (Analytic representation). An analytic representation is a functor from architecture schemes to a computable target: \( R : \mathrm{AATSch} \to \mathrm{Target} \).

The representative examples are:

R_graph  : AATSch -> Graph
R_matrix : AATSch -> Matrix
R_sig    : AATSch -> SignatureSpace
R_curv   : AATSch -> CurvatureValue
R_state  : AATSch -> StateAlgebra
R_trace  : AATSch -> RuntimeTrace
R_sem    : AATSch -> SemanticSpace
R_eff    : AATSch -> EffectSpace

Here AATSch is the category of AAT architecture schemes. More precisely, for a fixed reading parameter p, an object is a decorated architecture scheme X_AAT = (X, O_X, D_X), and a morphism f : X_AAT -> Y_AAT consists of an ordinary scheme morphism f : X -> Y together with compatibility data that preserves — or restricts as specified — the typed atom labels, the selected law reading, the obstruction ideals, the signature readings, and the interpretation maps. Fiber products are used when the underlying scheme fiber product exists and the decoration is coherently determined by pullback.

A representation sends a specific aspect of the architecture geometry to a computable form: 

architecture scheme
  -> graph
  -> matrix
  -> signature
  -> curvature
  -> trace

Principle VII.2.2 (Representation is a reading). A representation is a reading of the architecture geometry: a graph is a reading, a matrix is a reading, a signature is a reading, a curvature is a reading, a trace is a reading. Every representation is relativized to the fixed vocabulary, law universe, coverage topology, coefficient sheaf, and representation family: R(X) is a reading of X under the selected representation family.

3. Representation family

Definition VII.3.1 (Representation family). For an architecture geometry X, the representation family Rep(AAT) is a family of functors sending the AAT geometry to different target categories: 

Rep(AAT)
  =
  { R_graph, R_matrix, R_sig, R_curv, R_state, R_trace, ... }

The family reads one geometry from several angles: same geometry, many readings.

Example VII.3.2 (Main representations). The principal representations and what they read: 

graph representation:
  dependency, call, relation, service, context graph.

matrix representation:
  adjacency, incidence, transition, coupling matrix.

signature representation:
  selected signature axes.

curvature representation:
  obstruction valuation as curvature reading.

state representation:
  state transition algebra.

trace representation:
  runtime interaction trace.

semantic representation:
  language-game-relative semantic roles.

effect representation:
  effect and authority interactions.

These are the components of a period family that reads the same architecture geometry from different directions.

Definition VII.3.3 (Graph representation). The graph representation R_graph : AATGeometry -> Graph sends the selected relation data of the AAT geometry to the graph category. Representative edges are dependency, call, ownership, authority, state transition, and effect relations. Which relation becomes an edge is part of the representation profile.

Proposition VII.3.4 — Certified bounded inference

Acyclicity preservation

Lean: proved

Assume:

R_graph is selected for dependency-axis reading.
cycle obstruction witness is exact for the selected graph edges.
structural dependency obstruction is zero.

Then R_graph(X) is acyclic as the selected dependency graph.

Proof idea
Under witness exactness, any cycle in the selected graph reading would produce a selected cycle obstruction witness; zero structural dependency obstruction excludes every such witness, so no cycle exists.
Non-conclusion
Conversely, R_graph(X) being acyclic does not force the semantic, effect, or state transition obstructions to vanish. The inference runs on the selected dependency axis only.

Definition VII.3.5 (Matrix representation). Given a finite graph representation, its adjacency, incidence, or transition matrix is the matrix representation R_matrix(X). It is the linear-algebra form of the graph reading, and reads walk counts, rank, nilpotence, and spectral radius.

Proposition VII.3.6 — Certified bounded inference

Matrix walk reading

Lean: proved

Fix a finite adjacency matrix \( A_X = R_{\mathrm{matrix}}(X) \). Then \( (A_X^n)_{ij} \) reads the length-\( n \) walk count in the selected graph representation. When the selected graph is a finite directed acyclic graph, there exists \( N \) with \( A_X^N = 0 \).

Proof idea
The walk-count identity is induction on \( n \) along the standard recursion of counted walks over the adjacency relation. In an acyclic graph the vertex list of a counted walk has no repetition, so walk length is bounded by the vertex count and the cutoff \( N = \) card of the vertex type annihilates the matrix power.
Reading boundary
This is a reading of the finite relation structure of the graph representation; it reads no unselected coefficient, law, or semantic axis.

Principle VII.3.7 (Graph / matrix boundary). Graph and matrix are representations that read the selected relation of the AAT geometry. They give useful analytic readings, and they do not automatically read unselected coefficients, laws, or semantic axes.

4. Preservation and reflection

Definition VII.4.1 (Preservation properties). For a representation R: 

ZeroPreserving:
  structural zero -> analytic zero

ObstructionPreserving:
  structural obstruction -> analytic obstruction

ZeroPreserving means the representation reads a structural zero as zero; ObstructionPreserving means it reads a structural obstruction as an obstruction.

Definition VII.4.2 (Reflection properties). The reverse directions: 

ZeroReflecting:
  analytic zero + assumptions -> structural zero

ObstructionReflecting:
  analytic obstruction + assumptions -> structural obstruction

Reflection is stronger than preservation. To conclude a structural zero on the architecture geometry from a zero seen on the representation, coverage, witness completeness, axis exactness, and coefficient discipline are required: 

analytic zero
  + coverage
  + exactness
  + witness completeness
  -----------------------
  structural zero

Definition VII.4.3 (Conservative and faithful). A representation R is conservative, Conservative(R), when it reflects the selected isomorphisms, zeros, and obstructions; it is faithful, Faithful(R), when it distinguishes the selected morphisms. These properties grade the strength of a representation: 

weak reading
preserving reading
reflecting reading
conservative reading
faithful reading

Non-conclusion: an analytic zero taken alone, without the stated assumption package, licenses no structural-zero conclusion. The reflection predicates are usable exactly when the assumption package holds.

5. Period family

Definition VII.5.1 (Representation reading). For a representation R and an architecture geometry X, the reading is Read_R(X) = R(X). In the traditional broad sense this reading may be called a broad period: 

Period_R^{broad}(X)
  :=
Read_R(X)

Period_R(X)
  may be used as shorthand
  only when broad-period convention is fixed.

Representative readings: graph reading, matrix reading, signature reading, curvature reading, semantic reading, effect reading, runtime reading.

Definition VII.5.2 (Strict period). The strict period is a pairing with a cohomology class, cycle, trace, or operation loop, not a bare representation value. An obstruction cohomology class \( \omega \in H^n(X, \mathrm{Ob}_U) \) and a cycle or test object \( \gamma \in H_n(X, \mathbb{Z}) \) alone determine no scalar-valued period. Fix an additive period target \( \Lambda \) and a coefficient evaluation \( \mathrm{tr}_U : \mathrm{Ob}_U \to \underline{\Lambda} \), with induced map \( \mathrm{tr}_{U,*} : H^n(X, \mathrm{Ob}_U) \to H^n(X, \underline{\Lambda}) \). The strict obstruction period is

\[ \mathrm{Per}^{\mathrm{tr}_U}_{\omega,\gamma}(X) := \big\langle \mathrm{tr}_{U,*}(\omega),\ \gamma \big\rangle \in \Lambda. \]

The homology model \( H_n(X,\mathbb{Z}) \) and the target \( \Lambda \) are fixed by the ambient: a standard topological or analytic realization uses its singular homology; a finite context poset uses the order complex of the poset or the simplicial homology of the nerve of the chosen cover. The strict period is defined only when this homology model and the coefficient evaluation are fixed. When a monodromy representation \( \rho_U : \pi_1^{\mathrm{op}}(X) \to \mathrm{Aut}(F) \) is fixed, the trace period along an operation loop \( \gamma \) is \( \mathrm{Per}_\rho(\gamma) := \mathrm{Tr}(\rho_U(\gamma)) \).

Definition VII.5.2A (Finite poset / Čech homology model). For a finite context poset P = ArchCtx_min(A), the order complex Ord(P) has \( n \)-simplices the chains \( W_0 < W_1 < \dots < W_n \), and \( H_n^{\mathrm{poset}}(P,\mathbb{Z}) := H_n(C_*(\mathrm{Ord}(P),\mathbb{Z})) \). For a finite context cover 𝒰 = { W_i -> W }, the Čech nerve N(𝒰) distinguishes overlap components; its \( n \)-chains are the free abelian group on nonempty \( (n{+}1) \)-fold overlap components, with boundary the alternating sum of face restrictions \( \partial = \sum_j (-1)^j d_j \), and \( H_n^{\text{Čech}}(𝒰,\mathbb{Z}) := H_n(C_*(N(𝒰),\mathbb{Z})) \). For a cochain \( \omega \in C^n(𝒰, \mathrm{Ob}_U) \) and per-component coefficient evaluations \( \epsilon_{i_0 \dots i_n,\alpha} : \mathrm{Ob}_U(W_{i_0 \dots i_n,\alpha}) \to \Lambda \) compatible with face restriction, the pairing with a chain \( \gamma = \sum_\alpha m_\alpha\, W_{i_0 \dots i_n,\alpha} \) is

\[ \langle \omega, \gamma \rangle = \sum_\alpha m_\alpha\, \epsilon_{i_0 \dots i_n,\alpha}\big(\omega(W_{i_0 \dots i_n,\alpha})\big) \in \Lambda. \]

When \( \omega \) is a cocycle, \( \gamma \) is a cycle, and the coefficient evaluation is compatible with face restriction, this gives the cohomology-homology pairing over \( \Lambda \).

Example VII.5.2B (Pseudo-circle strict period). On the pseudo-circle boundary cover of Part IV: 

C_1 = Z e_sync direct_sum Z e_async
C_0 = Z v_0 direct_sum Z v_1
partial(e_sync)  = v_1 - v_0
partial(e_async) = v_1 - v_0

Hence \( \gamma = e_{\mathrm{sync}} - e_{\mathrm{async}} \) is a 1-cycle, and for the obstruction cocycle \( \omega = (r_{\mathrm{sync}}, r_{\mathrm{async}}) \),

\[ \mathrm{Per}^{\epsilon}_{\omega,\gamma} = \langle \omega, \gamma \rangle = r_{\mathrm{sync}} - r_{\mathrm{async}}, \]

and in particular \( \omega = (1,0) \) gives \( \mathrm{Per}^{\epsilon}_{\omega,\gamma} = 1 \). Throughout, three notions stay distinct: 

reading:
  R(X), the value under a representation.

broad period:
  a reading taken as a component of the period family.

strict period:
  a pairing with cohomology / cycle / trace / loop.

Definition VII.5.3 (Period family). The period family is Per(X) = { Read_R(X) | R in Rep(AAT) } — the many-faced analytic reading of the architecture geometry, summing graph, matrix, signature, curvature, semantic, effect, and runtime readings.

6. Period separation

Theorem VII.6.1

Period separation

Lean: proved (selected finite witness)

Architecture geometries X and Y can share the same broad period / reading under one representation while carrying different broad periods under another: 

Period_graph^{broad}(X) = Period_graph^{broad}(Y)

Period_semantic^{broad}(X) != Period_semantic^{broad}(Y)
  or
Period_effect^{broad}(X) != Period_effect^{broad}(Y)
Proof reading
The graph representation reads relation and dependency; the semantic representation reads meaning relativized to language games and use-rules; the effect representation reads authority, state, and runtime effect. These read different coordinate families, so equality in one coordinate family coexists with separation in another. Under the separation, each reading remains legitimate: graph is a legitimate broad period, semantic is another, effect is another.
Lean anchors
AAT.AG.RepresentationAnalysis.PeriodSeparationReadingFamily, PeriodSeparation, PeriodSeparationWitness.periodSeparation, RepresentationFamilyPeriodSeparationWitness, PeriodSeparationExample62.periodSeparation_theorem6_1 — proved under a selected finite witness.

Example VII.6.2 (Same graph, different semantic / effect reading). Take X and Y with the same directed graph reading: 

R_graph(X) = R_graph(Y)

components:
  A, B

edge:
  A -> B

The semantic readings differ:

R_sem(X):
  A uses B as query provider.

R_sem(Y):
  A uses B as command executor.

and the effect readings differ:

R_eff(X):
  A -> B is read-only.

R_eff(Y):
  A -> B may mutate shared state.

Hence the graph broad periods agree while the semantic and effect broad periods separate. This example is the concrete counterweight to any single-representation reading: a dependency diagram equality is a true statement about the graph window and only about the graph window. 7. Signature and curvature readings

Definition VII.7.1 (Signature reading). For a law universe U, the signature reading Sig_U(X) reads the architecture geometry along the selected signature axes. A selected axis being zero connects to the lawful locus of Part III, the obstruction cohomology of Part IV, and the law conflict of Part V: 

required signature zero
  reads
selected law failure vanishing.

Definition VII.7.2 (Curvature reading). The curvature reading kappa_U(X) reads the obstruction valuation geometrically. Under exactness, soundness, completeness, and coverage, zero curvature corresponds to factorization through the lawful locus: 

kappa_U(X) = 0
  corresponds to
X factors through Flat_U

Principle VII.7.3 (Curvature as reading). Curvature does not replace lawfulness; it is the analytic reading of the obstruction geometry. Lawfulness is factorization through Flat_U; curvature is a representation of the obstruction valuation.

8. Metric enrichment

Definition VII.8.1 (Metric AAT). Metric AAT is the structure of a metric enrichment on top of an AAT geometry: 

MetricAAT
  =
  AATGeometry
  + AtomMetricBundle
  + ConfigurationMetric
  + SignatureMetric
  + OperationCost
  + PathLength
  + HomotopyFillingCost
  + ObstructionMeasure
  + RepresentationMetric

The metric is enrichment, not ontology: metric is enrichment, distance is reading.

Principle VII.8.2 (Metric discipline). The metric enrichment obeys: 

distance does not create atoms
distance does not define lawfulness
distance does not remove obstruction cohomology
distance does not turn unmeasured into zero

Under this discipline, the metric works as a means of reading the architecture geometry — never as a source of architectural facts or lawfulness.

10. Distance to the lawful locus

Definition VII.10.1 (Operation distance). For architecture states or sections A and B, the operation distance d_op(A,B) is constructed from operation path costs, path lengths, and homotopy filling costs.

Definition VII.10.2 (Distance to flatness). The distance to the lawful locus is

\[ \mathrm{dist\_flat}_U(A) = \inf \,\lbrace\, d_{\mathrm{op}}(A, F) \mid F \in \mathrm{Flat}_U(X) \,\rbrace, \]

relativized to the range in which the DistanceValue domain or the selected cost domain possesses the infimum. It is the metric reading of the proximity of the state A to the lawful locus for U.

Principle VII.10.3 (Near flat is not flat). A small dist_flat_U(A) is a useful analytic reading. Flatness itself is the Part III factorization condition, defined as the vanishing of the obstruction ideal: 

Flat_U(A)
  iff
A factors through V(I_U)

Non-conclusion: proximity readings license no flatness claim. The factorization conclusion belongs to the ideal-theoretic layer, and in the Lean surface the implication dist_flat = 0 -> factorization is carried as a predicate to be supplied, never as an unconditional theorem.

11. Obstruction mass

Definition VII.11.1 (Obstruction measure). A measure is placed on the obstruction sheaf or the obstruction ideal sheaf: mu_Ob : Ob_U -> MeasureValue or mu_I : I_U -> MeasureValue.

Definition VII.11.2 (Obstruction mass). The obstruction mass of an architecture state A is read as

\[ \mathrm{Mass}_U(A) = \int_{X_A} \mu_{\mathrm{Ob}}, \]

the measure reading of the obstruction geometry — an analytic reading of the obstruction sheaf.

Principle VII.11.3 (Mass is support-relative). Obstruction mass is relativized to the measurement support: to coverage, coefficient, measure, and selected axes. Unmeasured regions and axes are never counted as zero mass.

12. Repair cost and route geometry

Definition VII.12.1 (Repair route). For an architecture state A and the lawful locus Flat_U(X), the repair route RepairRoute(A, Flat_U) is the family of operation paths from A toward Flat_U(X). With a fixed cost profile the minimal-cost route is read as argmin_P cost(P), where the cost may include path length, operation risk, boundary transfer, derived conflict, monodromy debt, and singularity contact.

Definition VII.12.2 (Repair profiles). Repair routes carry several readings: 

shortest repair:
  minimal operation cost.

safest repair:
  bounded side-effect and transfer.

structural repair:
  moves in normal cone direction toward vanishing of obstruction ideal.

stable repair:
  avoids increasing cohomology, derived conflict, monodromy debt.

Principle VII.12.3 (Shortest need not be best). The shortest repair is not always the best repair: 

shortest path
  may cross singular stratum.

cheap repair
  may create transferred obstruction.

metric improvement
  may miss normal cone direction.

Repair cost is relativized to both the metric reading and the derived geometry.

Definition VII.12.4 (Margin). Under a metric profile P, the margin of a state A is its distance to the selected unsafe boundary: for the selected safe region Safe_P with boundary BoundaryUnsafe_P,

\[ \mathrm{margin}_P(A) = \mathrm{dist}_P(A, \mathrm{BoundaryUnsafe}_P), \]

relativized to the selected measured axes, the metric, and the selected unsafe predicate.

Theorem VII.12.5 — Certified bounded inference

Margin stability

Lean: proved

Assume:

operation path p : A -> B is measured in P.
length_P(p) < margin_P(A).
dist_P satisfies the triangle inequality on the selected measured axes.
path endpoint distance satisfies dist_P(A,B) <= length_P(p).
margin_P(A) is the distance to the selected unsafe boundary.

Then B does not cross the selected unsafe boundary: \( \mathrm{length}_P(p) < \mathrm{margin}_P(A) \) implies that \( B \) remains inside the selected safe region.

Proof idea
Triangle inequality. If B lay on or beyond the selected unsafe boundary, then \( \mathrm{dist}_P(A,B) \ge \mathrm{margin}_P(A) \); but \( \mathrm{dist}_P(A,B) \le \mathrm{length}_P(p) < \mathrm{margin}_P(A) \), a contradiction.
Non-conclusion
The theorem asserts no safety for unmeasured axes or unselected law universes.

Definition VII.12.6 (Architectural Dehn function). Fix a presentation two-complex K_H(X,U) and a filling area reading. For a loop \( \ell \) with boundary length \( |\ell| \) and minimal filling area \( \mathrm{FillArea}(\ell) \), the architectural Dehn function is

Theorem VII.12.7 — Certified bounded inference

Definition VII.15.3 (U-detecting representation family). For a law universe U and a selected obstruction class family Obs_U, a representation family Rep is U-detecting when the vanishing of the selected readings detects selected witness vanishing: 

for alpha in Obs_U,
  (forall R in Rep, R(alpha) = 0)
  implies
  WitnessZero_U(alpha)

Here R(alpha) is the value of a cohomology class, obstruction ideal section, derived conflict class, or monodromy residue under the representation R, and WitnessZero_U(alpha) says every obstruction witness of alpha along the witness family chosen by U reads as zero. The property is placed on the family: a vanishing graph reading can coexist with a remaining semantic obstruction, so detection is relativized to the selected obstruction family and to the coverage / exactness assumptions.

Chapter summary

Part VII opens the geometry built in Parts I through VI through disciplined readings. Analytic representations are functors from decorated architecture schemes to computable targets (Definition VII.2.1); a representation family reads one geometry through many windows, with graph and matrix readings certified on the dependency axis (Propositions VII.3.4 and VII.3.6). Preservation, reflection, conservativity, and faithfulness grade the strength of a reading (Section 4). Broad periods are readings; strict periods are pairings of obstruction cohomology classes with cycles over a fixed homology model and coefficient evaluation, computed concretely on the pseudo-circle (Section 5). Period separation shows the same graph reading coexisting with different semantic and effect readings (Theorem VII.6.1). Signature and curvature read law axes and obstruction valuation (Section 7); metric enrichment, enriched distance values, distance to the lawful locus, obstruction mass, and repair route geometry read proximity and cost without ever turning unmeasured into zero (Sections 8–12), with margin stability and the observation gap lower bound as certified bounded inferences (Theorems VII.12.5 and VII.12.7). Singularity and monodromy acquire analytic profiles (Section 13); bounded analytic interpretation fixes the data every reading is relative to (Section 14); and under detection, adequacy, and exactness, a representation family is conservative for the selected obstruction family (Theorem VII.15.4). Theorem VII.16.1 packages the full tower — atom to analysis — as the algebraic-geometric AAT synthesis.

Part VIII asks under which finiteness, stability, functoriality, and verdict-discipline conditions these readings can be treated as measurements.

Terms

Analytic representation
A functor \( R : \mathrm{AATSch} \to \mathrm{Target} \) sending decorated architecture schemes to computable targets; a window on the geometry, not the geometry itself.
Representation family
Rep(AAT): the family of functors — graph, matrix, signature, curvature, state, trace, semantic, effect — reading one geometry from several angles.
Broad period / strict period
A broad period is a reading Read_R(X) under an explicit convention; a strict period is a pairing of a cohomology class with a cycle, trace, or loop over a fixed homology model, coefficient evaluation, and additive target.
Curvature reading
kappa_U(X): the obstruction valuation read geometrically; zero curvature corresponds to lawful-locus factorization under exactness, soundness, completeness, and coverage.
Distance value
A partially ordered enriched value object with measured, zero, unmeasured, unavailable, incomparable, and infinite; measured zero and unmeasured are never identified.
Distance to flatness
\( \mathrm{dist\_flat}_U(A) = \inf \,\lbrace\, d_{\mathrm{op}}(A,F) \mid F \in \mathrm{Flat}_U(X) \,\rbrace \): the metric reading of proximity to the lawful locus. Near flat is not flat.
Margin
margin_P(A): the distance of a state to the selected unsafe boundary; a measured path shorter than the margin keeps the endpoint inside the selected safe region (Theorem VII.12.5).
U-detecting family
A representation family whose all-readings-zero detects selected witness vanishing; under adequacy and exactness it is U-conservative for the selected obstruction family (Theorem VII.15.4).
