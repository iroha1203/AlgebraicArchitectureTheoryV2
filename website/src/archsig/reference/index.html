Primary workflow

analyze is the primary ArchSig workflow entry. It consumes archmap/v0.5.0, law-policy/v0.5.0, and a separate measurement-profile/v0.5.0, then writes the measurement packet, summary, insight report, viewer data, validation reports, and run manifest.

archsig-analysis-summary.json is the compact first reading surface. Read it together with archsig-measurement-packet.json, archsig-insight-report.json, archsig-atom-viewer-data.json, and archsig-run-manifest.json when source refs or measurement boundaries need inspection.

Workflow outputs

  • archmap-validation.jsonValidation report for the supplied archmap/v0.5.0.
  • law-policy-validation.jsonValidation report for the selected law-policy/v0.5.0.
  • normalized-archmap.jsonDeterministic finite-poset-site normalization of valid ArchMap input.
  • archsig-measurement-packet.jsonCurrent AG measurement handoff with structural verdicts, computed invariants, analytic readings, assumptions, and claim boundaries.
  • archsig-analysis-summary.jsonConclusion-first compact reading surface with findings, action queue, measurement basis, and boundaries.
  • archsig-insight-report.jsonPrioritized inspection queue and bounded reading guide derived from the measurement run.
  • archsig-insight-brief.mdHuman-readable brief for the same bounded insight report.
  • archsig-atom-viewer-data.jsonBounded ArchView projection that adds no structural verdict.
  • archsig-run-manifest.jsonGenerated artifact list, input digests, validation summary, and run metadata.
  • archsig-analysis-validation.jsonValidation report for the generated measurement packet.

Reference boundaries

  • Current input contractArchSig analysis consumes supplied archmap/v0.5.0, law-policy/v0.5.0, and measurement-profile/v0.5.0. The selected vocabulary, profile, and evidence determine what the run can report.
  • Comparison contractcompare computes record-level differences from two current run directories; it does not assign causality or semantic equivalence.
  • FieldSig handoffFieldSig reads archsig-measurement-packet/v0.5.0 as bounded current measurement state. It is not forecast truth, causal proof, or global safety.
  • Validation boundaryValidation checks artifact shape and guardrails. It is not a Lean proof, runtime correctness proof, or repair safety certificate.
